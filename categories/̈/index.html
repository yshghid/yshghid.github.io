                디스크 조각 모음 (부제: 생각 조각모음으로 지은 이유) #

                2025-01-31 #

                하드 디스크에 파일 저장 시 동시 작성, 동시 입출력, 저장 공간 부족 등 여러 요인으로 순차적으로 저장되지 않는 경우가 생긴다. 즉, 파일이 저장 장치 또는 운영 체제의 문제로 인해 여러 개로 쪼개져서 이곳 저곳에 흩어져서 저장되는데 이를 단편화(fragmentation)라고 한다.

                단편화가 발생한 파일을 읽기 위해서는 저장 장치가 여기저기 흩어진 조각들을 모두 찾아 읽어야 한다. 필요한 파일은 하나인데 막상 읽을 때는 더 작은 용량의 여러 파일을 읽는 효과가 발생하여 실행 속도가 느려진다. 디스크 조각 모음은 단편화된 파일을 한 덩어리로 모아준다.

                다까먹고 나서 복습하는 트랜스포머 #

                2025-02-01 #

                셀프 어텐션은 입력 단어 사이 연관성을 어텐션 점수로 계산하는 것. 입력 단어를 Q, K, V벡터로 변환하고, Q와 K벡터의 내적을 구하고, 소프트맥스를 적용하여 확률로 변환하고, V벡터를 이 확률로 가중합하여 최종 어텐션 점수를 얻는다. 단어 간의 관계는 여러 종류가 있는데 1개의 헤드를 사용하면 1가지 관계만 학습될것임. 멀티 헤드 어텐션을 수행해서 다양한 관계 패턴을 동시에 학습한다. 관계의 종류로는 문법적 관계(cat-sat은 주어-동사), 의미적 관계(cat-mat은 동물-사물이 놓인곳), 위치적 관계(on-mat)가 있다. 각 헤드에서 나온 결과를 병합하고 최종 선형 변환해서 하나의 벡터를 얻는다.

                특수상대성이론 설명 (별의 무덤을 본 사람들 읽다가 이해안돼서 적어봄) #

                2025-02-04 #

                특수상대성이론은 1905년 아인슈타인이 발표한 이론으로, 크게 두 가지 가정을 기반으로 한다. 상대성 원리는 모든 관성계(inertial frame, 등속 운동하는 좌표계)에서 물리 법칙은 동일하다는 것. 이는 갈릴레오 갈릴레이의 상대성 원리를 계승한 것으로, 특정한 절대적인 기준 좌표계(즉, ‘우주의 기준점’)가 존재하지 않는다는 뜻. 광속 불변의 원리는 빛의 속도(광속, c≈3.0×10^8m/s)는 진공에서 모든 관찰자에게 동일하게 측정된다는 것. 이것이 기존의 뉴턴 역학과 다른 핵심적인 차이점임.

                MHC molecules and TCRs: a numbers game #

                2025-02-08 #

                MHC(주조직적합성복합체)는 개별적으로 매우 다양한 유형을 가지고 있으며, 이는 개개인이 서로 다른 MHC 유형을 가질 가능성이 크다는 것을 의미한다. 인간의 MHC인 HLA 유전자는 현재 6,000개 이상의 변이를 보이며, 각각 특정한 아미노산 서열을 가진 펩타이드를 선호하는 경향이 있다. 예를 들어, HLA-DQ8은 P1과 P9 위치에서 글루탐산(G)을 선호한다. 이러한 특성으로 인해, 각 MHC 유형은 특정한 “펩타이드 바코드"를 가지게 되며, TCR이 이를 인식하고 면역 반응을 조절하는 역할을 한다. 결국, 개별적으로 제시할 수 있는 펩타이드 조합이 다르기 때문에 면역 반응에서도 개인차가 발생한다.

                Tensor Decompositions and Machine Learning #

                2025-02-09 #

                텐서는 다차원 배열로, 행렬의 개념을 일반화한 것이다. 이러한 텐서를 분해하는 기법인 텐서 분해는 비지도 학습과 다중 관계 데이터를 분석하는 데 유용하게 활용된다.

                기존의 행렬 분해는 다양한 문제 해결에 쓰이지만, 비고유성(non-uniqueness) 문제를 가지고 있다. 예를 들어, 행렬 M을 두 개의 행렬 A와 BT 로 분해할 때, 이러한 분해 방식이 유일하지 않을 수 있다. 심리학자 Charles Spearman은 1904년 인간의 지능을 두 가지 숨겨진 요인으로 설명할 수 있다고 주장했는데, 이를 입증하기 위해 학생들의 성적 데이터를 행렬로 저장하고 이를 행렬 분해를 통해 분석하고자 했다. 하지만, 행렬 분해의 비고유성 문제로 인해 명확한 요인을 도출하는 것이 어려웠다. 이러한 문제는 행렬 분해 시 발생하는 회전 문제(Rotation Problem)로도 설명될 수 있는데, 특정 행렬을 분해할 때, 임의의 가역 행렬을 곱하면 동일한 결과를 얻을 수 있어 고유한 해를 보장하지 못하는 경우가 많다.

                PARAFAC2 #

                2025-02-10 #

                PARAFAC(Parallel Factor Analysis)은 주성분 분석(PCA)을 다차원 데이터에 적용한 확장 모델로, 특히 Three-way 데이터(3차원 이상)를 분석하는 데 사용된다. 기본적인 PARAFAC 모델은 모든 데이터 샘플이 동일한 행과 열 개수를 가질 때 사용할 수 있지만, 현실 데이터에서는 샘플마다 행 개수가 다를 수 있다. 이러한 문제를 해결하기 위해 개발된 것이 PARAFAC2 모델이다.

                PARAFAC2 모델은 기존 PARAFAC과 달리 샘플마다 행 개수가 달라도 적용 가능하도록 설계되었다. 이를 위해 행렬 Pk를 도입하여 샘플별 행 정보를 유지하면서도 모델을 최적화할 수 있도록 한다. 즉, 모든 샘플이 공유하는 로딩 행렬 A를 유지하면서도, 샘플별로 다른 행렬 Pk를 적용하여 개별적인 행 정보도 반영하는 방식이다.
