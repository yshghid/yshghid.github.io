위 코드에서는 통계 테스트로 'lrt'를 사용하였는데 이 경우 차등 발현의 크기가 제공되지 않음. 차등 발현의 크기도 제공되는 test type을 사용하려면 ward test 사용하기. In the above code, 'lrt' was used for statistical testing, but in this case, the magnitude of differential expression is not provided. To use a test type that also provides the magnitude of differential expression, use the ward test.

```
#sleuth_prep까지 동일 #same until sleuth_prep

so <- sleuth_fit(so, ~condition, 'full')
so <- sleuth_fit(so, ~1, 'reduced')
so <- sleuth_wt(so, 'condition150')
sleuth_table_gene <- sleuth_results(so, 'condition150', 'wald', show_all = FALSE)
sleuth_save(so, './sleuth_ward/150_con_so.sleuth')
write_csv(sleuth_table_gene, './sleuth_ward/150_con.csv')
```

### fastq 파일 길이 확인 Checking the length of fastq file
```python
import os

path='./Bowtie2Index'
fastq_list = [file for file in os.listdir(path) if file.endswith(".fastq")]
fastq_list.sort()

for fastq in fastq_list:
    fastq_path=path+f'/{fastq}'
    with open(fastq_path, 'r') as file:
        lines = [line.rstrip('\n') for line in file]
        print(fastq)
        print('read counts: ',(len(lines)+3)//4)
        for idx,i in enumerate(lines):  
            if (idx+1)%2==0 and len(i) !=101:
                print(f'line {idx+1}(4n+{(idx+1)%4})')
                print(f'length : {len(i)}')
```
