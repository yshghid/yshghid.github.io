### README

"Tools"의 코드들은 Bioinformatics 툴의 파이프라인이 별도의 데이터 다운로드 없이 돌려 볼 수 있도록 짜여져 있다. 블로그에 작성된 코드는 약간의 축약 버전이므로, 실제로 돌려 보려면 [깃허브][1]에 올라가있는 code.ipynb 를 사용하면 된다. 



Kallisto
-------------


```
$ git clone https://github.com/pachterlab/kallisto.git

$ kallisto index -i transcripts.idx homo_sapiens.tar.gz

$ kallisto quant -i transcripts_cDNA.idx -o output -t 40 -b 100 pair_1.fastq pair_2.fastq
```
