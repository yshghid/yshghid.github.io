자주 쓰는 커맨드 모음 Command Archieve

백그라운드로 돌리기 Running in background
-------------

```
tophat -o ./tophat_out_con-3 hg38.fa 5-AZA_con-3_1.fastq 5-AZA_con-3_2.fastq & 
bg
disown-h

# tophat 사용해서 RNA-seq 얼라인 할 때 시간 너무 오래걸려서 썼다.
# 터미널을 끄거나 맥북 덮어도 진행 되고, 터미널에 진행 상태는 뜨지 않음. 
# nohup이 보통 쓰는거라고 하는데 이상하게 안먹어서 썼다.
```

서버 명령어 Server 
-------------

```
# 실행중인 프로세스 확인 Checking the currently running processes
ps -ef|grep "tophat_out_33-1"
ps -u ysh980101
```

```
# 세션 죽이기 Killing the session
kill -9 세션번호
```

티뮥스 tmux
-------------


```
# 세션 종료 End session
$ Ctrl + d
```
```
# 세션 이름 지정하기 Assign session name
$ tmux new -s [세션명]
```
```
# 세션 종료하지 않고 기존 shell로 이동 Move to the previous shell without ending the session
$ (Ctrl + b) + d
```
```
# 세션 불러오기 Retrieve session
$ tmux ls
$ tmux attach -t [세션명]
```
