    [Note] Genomic regions

    July 28, 2024
    Note

    TSS1500 (Transcription Start Site 1500) 유전자의 전사 시작 위치(TSS)로부터 최대 1500 염기쌍(bp) 상류(upstream) 영역. 프로모터(promoter) 영역의 일부로 간주되며, 유전자 발현의 조절에 중요한 역할을 함. 전사 인자와 같은 조절 단백질들이 이 영역에 결합하여 전사의 개시를 조절할 수 있음. TSS200 (Transcription Start Site 200) 유전자의 전사 시작 위치(TSS)로부터 200 염기쌍(bp) 이내의 상류(upstream) 영역. TSS1500보다 더 프로모터에 가깝고, 전사 조절에서 더욱 중요한 역할을 할 수 있음. TSS200 영역에는 전사 인자 결합 사이트와 같은 중요한 조절 요소들이 위치할 수 있음. ...
