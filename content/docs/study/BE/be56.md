# bash #1 venv 가상환경 설정

#2026-02-08

---

스칼라 하면서 개조돼서 이젠 conda보다 venv가 훨씬 익숙해져버렸다 ㅠ

```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt
```
