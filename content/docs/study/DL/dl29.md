#1 SHAP Dot Plot — 각 사이토카인이 어떻게 기여하는가

이 그림은 이전 단계에서 계산한 SHAP 값을 가장 직관적으로 보여주는 방법이다. 상위 20개 사이토카인을 세로축에 놓고(위에 있을수록 더 중요하다), 각 샘플의 SHAP 값을 가로축에 점으로 찍는다. 가로축의 오른쪽은 양수(중증 확률을 올리는 방향)이고 왼쪽은 음수(중증 확률을 낮추는 방향)다.여기서 핵심적인 추가 정보가 색상이다. 각 점의 색깔이 해당 샘플에서 그 사이토카인의 실제 발현량을 나타낸다. 빨간색이면 발현이 높은 샘플이고 파란색이면 발현이 낮은 샘플이다.

```plain text
← SHAP 음수           0          SHAP 양수 →
  (중증 확률 감소)                (중증 확률 증가)

                    발현 높음(빨강) ●────────────→
CD274   ─────────────────────●●●●●●●●●●●●●●●
         ←○○○ (발현 낮음=파랑)

KIT     ──────────────────●●●●●●●●●●
FLT3LG  ──────────────────●●●●●●●●
GDF15   ─────────────────●●●●●●●
...

X축: SHAP 값 (기여도 크기 및 방향)
점 색깔: 발현 수준 (빨강=높음, 파랑=낮음)
Y축: 사이토카인 (SHAP 중요도 순, 위가 더 중요)
```

이 두 정보(SHAP 값의 위치와 점의 색깔)를 함께 읽으면 사이토카인의 작동 방향을 한눈에 파악할 수 있다. CD274을 보자. 빨간 점(발현 높음)이 오른쪽(SHAP 양수, 중증 확률 증가)에 몰려 있고, 파란 점(발현 낮음)이 왼쪽(SHAP 음수, 중증 확률 감소)에 몰려 있다. 이건 "CD274이 높으면 중증 확률이 올라가고, 낮으면 내려간다"는 뜻이다. 발현량과 중증도가 같은 방향으로 움직인다. 직관적으로 말이 된다. CD274(PD-L1)은 면역 억제 분자이므로 높을수록 면역 기능이 저하되어 중증화 위험이 커진다.

TNFSF10을 보면 패턴이 뒤집힌다. 빨간 점(발현 높음)이 왼쪽(SHAP 음수)에 있고, 파란 점(발현 낮음)이 오른쪽(SHAP 양수)에 있다. "TNFSF10이 높으면 오히려 중증 확률이 내려가고, 낮으면 올라간다"는 뜻이다. 4단계에서 분석한 것과 정확히 일치한다. TNFSF10(TRAIL)은 감염 세포를 제거하는 무기이므로, 이것이 부족할 때 중증화가 진행되는 것이다.

```plain text
사례 1: CD274
  빨간 점이 오른쪽에 집중 → "CD274가 높으면 중증 확률 올라감"
  파란 점이 왼쪽에 집중 → "CD274가 낮으면 중증 확률 내려감"
  → 양방향 일관적: CD274 발현량과 중증도가 비례

사례 2: TNFSF10 (Step 4에서 ΔRank +54)
  빨간 점이 왼쪽에 → "TNFSF10이 높으면 오히려 중증 확률 내려감"
  파란 점이 오른쪽 → "TNFSF10이 낮으면 중증 확률 올라감"
  → 역방향: TNFSF10 부재가 중증의 특징
```

이 그림 하나로 166개 사이토카인 중 가장 중요한 20개가 무엇인지, 각각이 어떤 방향으로 기여하는지, 발현량과 기여도의 관계가 일관적인지를 모두 확인할 수 있다.

    max_display=20,          # 상위 20개만 표시
    cmap='bwr',              # 파랑-흰색-빨강 색상

#2 SHAP Clustermap — 전체 패턴을 조감하기

## 그림 2: SHAP Clustermap (논문 보충 그림 S3)

Dot Plot이 상위 20개 사이토카인에 집중했다면, Clustermap은 전체 사이토카인 × 전체 샘플의 SHAP 패턴을 한 그림에 담는다.

이 그림은 히트맵이다. 가로축이 404개 환자 샘플이고 세로축이 사이토카인이다. 각 칸의 색깔이 해당 샘플에서 해당 사이토카인의 SHAP 값을 나타낸다. 빨간색이면 양수(중증 확률 증가 기여)이고 파란색이면 음수(중증 확률 감소 기여)다.
핵심은 계층적 클러스터링이 적용된다는 점이다. 히트맵의 행과 열이 비슷한 것끼리 가까이 재배열된다. SHAP 패턴이 비슷한 사이토카인들이 세로축에서 이웃하게 되고, SHAP 패턴이 비슷한 샘플들이 가로축에서 이웃하게 된다.

```plain text
         ┌────────────── 샘플 (열) ────────────────┐
         │ [계층적 클러스터링으로 정렬]              │
         ├──── 색상 바 1: PPG ────────────────────┤
         │ 주황=mDP  초록=mRP  빨강=sDP  보라=sRP  │
         ├──── 색상 바 2: Severity ───────────────┤
         │ 주황=중등도  빨강=중증                   │
         │                                        │
사이토카인 │  [SHAP 히트맵]                          │
  (행)    │  빨강: SHAP 양수 (중증 예측 증가에 기여)  │
         │  파랑: SHAP 음수 (중증 예측 감소에 기여)  │
         └────────────────────────────────────────┘
```

계층적 클러스터링의 원리는 단순하다. 가장 비슷한 두 사이토카인을 찾아서 묶는다. 묶인 것과 가장 비슷한 다음 사이토카인을 또 묶는다. 이걸 반복하면 나무(덴드로그램) 구조가 만들어진다. 가지가 가까운 사이토카인들은 패턴이 비슷하고, 가지가 먼 것들은 패턴이 다르다.

```plain text
비슷한 것들끼리 모으는 순서:

1. 가장 비슷한 두 사이토카인을 묶음 (CD274 ↔ FLT3LG)
2. 묶인 것과 가장 비슷한 다음을 묶음 (CD274+FLT3LG ↔ KIT)
3. 반복 → 나무(덴드로그램) 구조 형성

결과:
  히트맵의 행(사이토카인)과 열(샘플)이
  비슷한 것들끼리 가깝게 재배열됨
  → 패턴 블록이 뚜렷하게 드러남
```

재배열된 히트맵에서는 패턴 블록이 뚜렷하게 나타난다. 가로축 위에 색상 바가 두 줄 붙어 있다. 첫 번째 줄은 PPG 레이블(주황=mDP, 초록=mRP, 빨강=sDP, 보라=sRP)이고, 두 번째 줄은 중증도(주황=중등도, 빨강=중증)다. sDP 샘플들(빨간 색상 바) 아래를 보면 특정 사이토카인 행들이 일관되게 빨간색(SHAP 양수)이고, 다른 행들은 일관되게 파란색(SHAP 음수)인 블록이 보인다. 이 블록이 "중증 악화 단계에서 올라가는 사이토카인들"과 "내려가는 사이토카인들"을 시각적으로 보여준다.

```plain text
sDP 샘플(빨강 색상 바) 아래에서:
  특정 사이토카인 행들이 일관되게 빨간색 → sDP 특이적 고발현
  다른 사이토카인 행들이 일관되게 파란색 → sDP에서 낮음(예: TNFSF10)

→ "중증 악화 단계에서 어떤 사이토카인이 올라가고 내려가는가"
  전체 패턴을 한눈에 파악
```

Dot Plot이 개별 사이토카인의 이야기를 들려준다면, Clustermap은 전체 면역 프로파일의 조감도를 제공한다.

```python
from sklearn.preprocessing import StandardScaler

# SHAP 값 정규화
scaler = StandardScaler()
df_scaled = pd.DataFrame(
    scaler.fit_transform(patient_shap),
    index=patient_shap.index,
    columns=patient_shap.columns
)

# 컬러 바 (PPG, 중증도)
col_ppg = patient_meta['Detailed_PPG'].map({
    'sDP': 'red', 'mDP': 'orange', 'sRP': 'purple', 'mRP': 'green'
})

g = sns.clustermap(
    df_scaled.T,           # 전치: 사이토카인이 행, 샘플이 열
    cmap='coolwarm',       # 파랑-흰색-빨강
    col_colors=col_colors, # 상단 색상 바
    vmin=-3, vmax=3,       # SHAP 값 범위
)
```

#3 UMAP Scatter Plot — 면역 패턴의 지도

4단계에서 UMAP으로 166차원을 2차원으로 압축했다. 이 2차원 좌표에 점을 찍으면 면역 패턴이 비슷한 샘플들이 가까이 모여 있는 지도가 된다.

이 그림은 두 개의 산점도를 나란히 놓는다. 왼쪽은 PPG 레이블로 색을 칠한다. mDP는 주황, mRP는 초록, sDP는 빨강, sRP는 보라다. 오른쪽은 같은 점들을 DBSCAN 클러스터 번호로 색을 칠한다.

```plain text
왼쪽 그림: PPG 색상             오른쪽 그림: DBSCAN 클러스터 색상
 ·  ·                           ·  ·
  ·  ·  ← sDP(빨강)              ·  ·  ← 클러스터 3
   · ·                             · ·
      · ·                             · ·
        · ·  ← mDP(주황)              · ·  ← 클러스터 1, 2
       · · ·                          · · ·
    · · ·  ← mRP(초록)             · · ·  ← 클러스터 0
 · ·                            · ·  ← 클러스터 5
  ·  ←  sRP(보라)                 · ← 노이즈(-1, 회색)
```

왼쪽 그림에서 같은 색 점들이 한 곳에 모여 있는지를 본다. sDP(빨강) 점들이 한쪽 구석에 모여 있고, mRP(초록) 점들이 반대편에 모여 있다면, PPG 레이블이 면역 패턴의 실제 차이를 반영한다는 시각적 증거다. 만약 네 가지 색이 뒤죽박죽 섞여 있다면, PPG 레이블링이 사이토카인 패턴과 무관하다는 뜻이므로 전체 분석의 기반이 흔들린다.

오른쪽 그림과 비교하면 DBSCAN 클러스터와 PPG의 관계가 보인다. 클러스터 3이 왼쪽 그림의 sDP(빨강) 영역과 겹치면, 클러스터 3은 중증 악화 샘플이 주로 속한 군집이다. 클러스터 0이 mRP(초록) 영역과 겹치면, 이 클러스터는 중등도 회복 샘플의 군집이다. 4단계에서 통계 분석에 사용할 클러스터를 선택한 근거가 이 그림에서 시각적으로 확인된다.

```plain text
클러스터 3이 sDP(빨강) 위치와 겹침
→ 클러스터 3은 sDP 특화 클러스터

클러스터 1, 2가 mDP(주황) 위치와 겹침
→ 클러스터 1, 2는 mDP 특화

→ sDP vs sRP 분석: 클러스터 3 주변 선택
  mDP vs mRP 분석: 클러스터 1, 2, 0 주변 선택
```

```python
fig, axes = plt.subplots(1, 2, figsize=(18, 7))

# 왼쪽: PPG 색상
PPG_COLORS = {'mDP': 'orange', 'mRP': 'green', 'sDP': 'red', 'sRP': 'purple'}
for ppg, color in PPG_COLORS.items():
    mask = patient_meta['Detailed_PPG'] == ppg
    axes[0].scatter(
        umap_df.loc[mask, 'UMAP_1'],
        umap_df.loc[mask, 'UMAP_2'],
        c=color, label=ppg, s=40, alpha=0.7
    )

# 오른쪽: DBSCAN 클러스터 색상
cmap = plt.cm.get_cmap('tab20', 11)
for cluster in range(11):
    mask = umap_df['Cluster'] == cluster
    axes[1].scatter(
        umap_df.loc[mask, 'UMAP_1'],
        umap_df.loc[mask, 'UMAP_2'],
        c=[cmap(cluster)], label=f'Cluster {cluster}', s=40, alpha=0.7
    )
```

#4 Volcano Plot — 통계 유의성과 발현 방향을 동시에

Volcano Plot은 통계 분석 결과를 가장 효율적으로 보여주는 시각화다. 가로축은 평균 발현 차이(DP - RP)이고, 세로축은 통계 유의성(-log10(보정된 p-value))이다. 세로축에 로그를 씌우는 이유는 p-value가 극도로 작은 값(예: 2.70 × 10⁻¹³)이기 때문이다. 로그를 씌우면 이런 극단적 값도 그래프에 깔끔하게 표현된다.

```plain text
-log10(P-adj)
    ↑
  8 ┤         CD274●              CD274●
    ┤        FLT3LG●             FLT3LG●
  6 ┤     S100A12●   CXCL10●
    ┤      LGALS9●     GDF15●
  4 ┤       VCAM1●      ERBB2●
    ┤
  2 ┤  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  ← p=0.05 기준선
    ┤         회색 점들 (비유의)
  0 └─────────────────────────────────────▶ 평균 발현 차이 (DP - RP)
   음수(RP에서 높음)    0    양수(DP에서 높음)

     ← RP에서 더 높음        DP에서 더 높음 →
       (빨강=유의)            (파랑=유의)
```

이 그림이 Volcano(화산)라고 불리는 이유는 모양 때문이다. 대부분의 사이토카인은 차이가 작고 유의하지 않아서 그림의 아래쪽 가운데에 모여 있다. 유의한 사이토카인들은 위쪽으로 솟아오르면서 좌우로 퍼진다. 화산이 분출하는 모양과 비슷하다.
가로축의 오른쪽에 있으면서 위쪽에 있는 점(빨간색으로 표시)은 DP에서 RP보다 유의하게 높은 사이토카인이다. 악화 단계를 특징짓는 사이토카인이다. 왼쪽에 있으면서 위쪽에 있는 점(파란색)은 RP에서 더 높은 사이토카인이다. 회복 단계를 특징짓는다. 수평 점선은 보정된 p = 0.05 기준선이다. 이 선 아래의 회색 점들은 통계적으로 유의하지 않은 사이토카인들이다.

```plain text
오른쪽 위 (빨강 점): DP에서 더 높고 유의한 사이토카인
  → 악화 단계를 특징짓는 사이토카인
  예: CD274, FLT3LG, PTX3, CXCL10

왼쪽 위 (파랑 점): RP에서 더 높고 유의한 사이토카인
  → 회복 단계를 특징짓는 사이토카인

가운데 (회색 점): 유의하지 않음 (FDR > 0.05)
```

핵심은 mDP vs mRP Volcano와 sDP vs sRP Volcano를 나란히 비교하는 것이다. 중등도 Volcano에서는 아래쪽에 묻혀 있던 점(비유의)이 중증 Volcano에서는 위쪽으로 솟아오르면(유의해지면), 그 사이토카인이 중증에서 새롭게 중요해진 것이다. TNFSF10이 정확히 이 패턴을 보인다. 중등도 Volcano에서는 기준선 아래의 회색 점이었는데, 중증 Volcano에서는 상위권의 색깔 점으로 올라온다. MPO, HGF도 마찬가지다. 이 시각적 비교가 다음 그림인 ΔRank Chart의 직관적 근거를 제공한다.

```plain text
mDP vs mRP Volcano:               sDP vs sRP Volcano:

높음                                높음
  CD274 ●                            CD274 ●
  FLT3LG ●                           FLT3LG ●
  S100A12 ●                          CXCL10 ●
  LGALS9 ●                           TNFSF10 ● ← 여기서 처음 등장!
                                     GDF15 ●
  TNFSF10은 낮음(비유의)              TNFSF10은 높음(유의)
  MPO는 낮음(비유의)                  MPO는 높음(유의)
  HGF는 낮음(비유의)                  HGF는 높음(유의)

→ ΔRank가 크게 양수인 사이토카인들이
  sDP vs sRP Volcano에서 새롭게 위로 올라옴
```

```python
def plot_volcano(results_df, title, label1, label2, ax):
    df = results_df.copy()
    df['-log10_padj'] = -np.log10(df['P_adj'].clip(1e-15))

    # 색상 분류
    colors = df.apply(lambda row:
        'gray'      if row['P_adj'] >= 0.05 else
        'firebrick' if row['Mean_Diff'] > 0  else  # DP 높음
        'steelblue',                                # RP 높음
        axis=1
    )

    ax.scatter(df['Mean_Diff'], df['-log10_padj'],
               c=colors, alpha=0.7, s=50)
    ax.axhline(-np.log10(0.05), color='black', linestyle='--')  # 기준선
    ax.axvline(0, color='black', linewidth=0.5)
```

#6 핵심 사이토카인 Boxplot 

마지막 그림은 4단계에서 발견한 네 가지 핵심 사이토카인(CXCL10, TNFSF10, GDF15, PTX3)의 실제 발현 분포를 PPG별로 보여주는 상자 그림이다.

상자 그림은 분포를 압축적으로 보여준다. 상자의 가운데 선이 중앙값, 상자의 위아래 끝이 사분위수(25%~75% 범위), 수염이 전체 범위, 그리고 개별 점들이 실제 데이터다. 네 가지 PPG(mDP, mRP, sDP, sRP)가 나란히 놓여서 직접 비교할 수 있다.

```plain text
CXCL10 (케모카인, JAK-STAT):
         mDP    mRP    sDP    sRP
발현   ───────────────────────────
상위   │  │  │   │  │   │  │   │  │
중앙   │──┼──│   │──┼──│   │──┼──│   │──┼──│
하위   │  │  │   │  │   │  │   │  │
       중간    낮음   높음★  중간
                      ↑ sDP에서 가장 높음
                        (사이토카인 폭풍 지표)

TNFSF10 (TRAIL, NF-κB):
         mDP    mRP    sDP    sRP
발현   ───────────────────────────
       중간    중간   낮음★  중간
                      ↑ sDP에서 가장 낮음!
                        (선천 면역 약화)

GDF15 (스트레스, MAPK):
         mDP    mRP    sDP    sRP
발현   ───────────────────────────
       중간    낮음   높음★  중간
                      ↑ sDP에서 최고
                        (조직 손상 반응)

PTX3 (급성기 반응, NF-κB):
         mDP    mRP    sDP    sRP
발현   ───────────────────────────
       높음    중간   최고★  중간
              ↑ 전반적으로 DP > RP

```

CXCL10을 보면 sDP에서 상자가 가장 높이 올라가 있다. 중증 악화 단계에서 이 염증성 케모카인이 가장 높다는 뜻이다. 면역세포를 불러모으는 신호가 폭주하고 있는 상태다. TNFSF10을 보면 반대 패턴이다. sDP에서 상자가 가장 낮다. 다른 세 그룹에서는 비슷한 수준인데 sDP에서만 유독 떨어진다. 감염 세포를 제거하는 무기가 중증 악화에서만 특이적으로 약해져 있다. GDF15는 sDP에서 가장 높다. 조직 손상 스트레스 반응이 극에 달해 있다. PTX3도 sDP에서 가장 높다. 급성 염증 반응이 최고조에 있다.

이 네 개의 상자 그림을 함께 읽으면 하나의 서사가 완성된다. 중증 악화(sDP) 환자에서는 감염 세포를 제거하는 TRAIL 메커니즘이 무너져 있고(TNFSF10↓), 그 결과 면역세포를 소집하는 신호가 폭주하고(CXCL10↑), 과잉 면역 반응으로 조직이 심각하게 손상되고(GDF15↑), 급성 염증이 극에 달한다(PTX3↑). 이건 숫자 테이블에서는 보이지 않던 이야기다. 그림이 이야기를 보이게 만든다.

```plain text
정상 면역 반응 시나리오:
  바이러스 침입
    → TNFSF10(TRAIL) 상승: 감염 세포 제거  ← 선천 면역 정상 작동
    → 적절한 CXCL10: 면역세포 모집
    → GDF15, PTX3 상승 후 빠른 정상화
    → 회복

  바이러스 침입
    → TNFSF10(TRAIL) 낮음: 감염 세포 제거 실패  ← 선천 면역 약화
    → CXCL10 폭증: 과도한 면역세포 모집 (사이토카인 폭풍)
    → GDF15, PTX3 극도로 상승: 심각한 조직 손상
    → 악화 → 기계 환기 → 중증화
```

#7 정리

이 여섯 그림은 무작위로 나열된 것이 아니라 하나의 추론 사슬을 따라 배치되어 있다.
SHAP Dot Plot이 먼저 "어떤 사이토카인이 중요하고 어떤 방향으로 기여하는가"를 밝힌다. SHAP Clustermap이 "이 패턴이 전체 샘플에서 PPG와 일관되게 나타나는가"를 확인한다. UMAP Scatter가 "비슷한 패턴의 샘플들이 실제로 공간적으로 군집을 이루는가"를 보여준다. Volcano Plot이 "악화 단계와 회복 단계 사이에 통계적으로 유의한 차이를 보이는 사이토카인은 무엇인가"를 검증한다. ΔRank Chart가 "중등도에서 중증으로 넘어갈 때 새롭게 중요해지는 사이토카인은 무엇인가"라는 핵심 발견을 제시한다. 마지막으로 Boxplot이 "핵심 사이토카인들의 실제 발현 패턴이 생물학적으로 말이 되는가"를 확인한다.

```plain text
SHAP Dot Plot              SHAP Clustermap
(어떤 사이토카인이 중요?)    (전체 패턴이 PPG별로 다른가?)
      ↓                           ↓
      └──────────┬────────────────┘
                 ↓
          UMAP Scatter
     (비슷한 샘플들이 모이는가?)
                 ↓
          DBSCAN 클러스터
                 ↓
     Volcano Plot (mDP vs mRP / sDP vs sRP)
     (어떤 사이토카인이 DP/RP를 구분하는가?)
                 ↓
           ΔRank Chart
     (중증도에 따라 중요도가 어떻게 달라지는가?)
                 ↓
      Key Cytokine Boxplot
     (4개 핵심 사이토카인의 실제 발현 패턴 확인)
```

각 그림은 이전 그림에서 제기된 질문에 답하면서 새로운 질문을 던지고, 다음 그림이 그 질문에 답하는 구조다. 그리고 마지막 Boxplot에서 모든 분석 결과가 하나의 생물학적 서사로 수렴한다. SHAP이 "왜 이 샘플이 중증인가"를 분해했고, UMAP과 DBSCAN이 분석 단위를 정리했고, 통계 검정이 엄밀성을 보장했고, ΔRank가 중증 특이적 바이오마커를 발굴했으며, Boxplot이 그 바이오마커의 실제 행동을 확인했다. 수만 개의 숫자로 시작해서 "TRAIL이 약하면 사이토카인 폭풍이 온다"는 한 문장의 결론으로 도달하는 여정이다.
