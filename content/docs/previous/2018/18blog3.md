# 결혼

#2018-02-04 00:09:25

---

어제 인터넷 서핑을 하다가 읽은 글에서 첫 연애는 24~25세에 하는것이 가장 좋다고 했다. 그정도 되고 나면 아무래도 사람이 인성적으로(?) 좀 안정되기도 하고 자기자신이 확립된 시기니까 그런가보다. 무슨 말인지는 알겠어 흑역사 만들일 줄어든다는거잖아ㅋㅋ(엄청 중요) 근데 이건 연애뿐만이 아니라 사람만나는 모든것에 해당하는 것 같다.

드라마 내일 그대와 에서 이제훈이 결혼은 평생을 약속한 하우스메이트라는 이야기를 했댜. 나는 결혼을 하지않고 살고싶다 주의인데 약간 비슷한 입장인게 그냥 말그대로 집을 공유하고 저녁과 밤 시간을 공유하는 특별한 사이 정도라면 좋은것같다는 말이다
