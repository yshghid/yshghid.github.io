                              밥은침대에서 먹어야 제맛이고 아침햇살은 암막커튼 안쳐놓은곳에서만 들어와야 제맛이로다 극한의 평화 먹고싶은걸 먹고싶은 곳에서 먹을수만 있어도 행복한데.. 사진에는 없지만 노브랜드 오리지널 포테이토칩 말고도 폴바셋 쿠크다스가있는데 요즘 내인생 동반자이다 오리지널 포테이토칩 맛있긴한데 이마트에서 요즘 품절상태다 ㅜ 그리스 베리아 진짜평화롭네 ㅋㅋ 예뻐서 남겨놓음 내 예쁜 면허증 ㅎㅎ 오늘은 길혜민이랑 만나서 카공했는데 되게행복하진 않았지만 그런대로 괜찮은 하루였다 밤에는 좀 오래안울어서인지 속이꽉막혀있었다 그래서 당신의특별한우울 오랜만에 꺼내서 읽었다. 근데 공부했더니 책이 더 잘읽히네 조금더노력하면 점점나아질것만같아 희미한희망이 보였다 그리고 시카고의 잠못드는밤 영화 진짜 구리다 ㅋㅋ 제목은 너무예뻤는데... 아기가 예뻐서 끝까지 다봄 재미는진짜없음
