                생각많은밤 #

                #2017-05-01 01:04:13

                여중생A를 정주행했다 잊고있던 웹툰 몇십편을 몰아보는것은 진정한 행복이라고 생각해 (아님)(그래세상엔더좋은일이많겠지)

                별로 한게없군. 글을 쓰지도 물건을사지도 뭔가를 구상하지도않고 그렇게 4월은 흘러갔다

                오늘 롤까스를 먹었는데 맛있었어 난 지금 배가고프지만 잠이 몰려오는 새벽에 영화를 보는것같은 쾌감적인 고통이라서 뭐안먹고 그냥 잘것이다. 그리고 나는 항상 이겨야한다는 압박은 좀 떨치고 보통의 미학을 추구하는 사람이 되면좋겠으니까 내일부터 그렇게 살아야지 말했잖아 내가 가진것은 실천력과 행동력뿐…

                언제나 최상에 만족하려니까 그무엇도 사랑할수가없더라 그게다야사실 ㅎㅎ
