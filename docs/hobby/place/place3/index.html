                              브라우니맛집 서촌스코프 브이랩커피

                              브라우니맛집 서촌스코프 브이랩커피 #

                              #2026-07-07

                              서촌 놀러가서 사먹었던 서촌스코프 브라우니!

                              맛은 스코프브라우니, 헤이즐넛비스킷, 라즈베리크림치즈 이렇게 3개 있고 스코프랑 헤이즐넛맛을 먹어봤는데 겉은 바스락하고 속은 쫀쫀한 완전 맛있는 브라우니였다 ㅎㅎ 특히 헤이즐넛은 투머치일줄 알았는데 너무너무 잘어울려서 오히려 플레인보다 더 맛있었다. 설명을 보니까 벨기에 초콜릿과 발로나 코코아파우더가 들어갔다고 한다.

                              버터 쇼트브레드 비스킷도 사봤는데 스콘처럼 고소한 빵-과자 사이 식감일줄 알았는데 의외로 잘 바스라지는 식감이었다. 맛도 버터리한 밀가루 빵의 맛을 생각했었는데 슈가파우더 느낌의 단맛이 훨씬 강한 달달한 디저트여서 커피보다 티 종류랑 잘 어울릴 맛이었다 ㅎㅎ

                              #

                              두번째는 배달로 시켜먹은 브이랩커피 브라우니! 맛있는 브라우니를 먹고싶어서 많이 찾아보다가 아래 리뷰를 보고 달달+꾸덕일것같아서 시켜봤다.

                              맛은 기대한대로 완전 꾸덕하고 달달했다 ㅎㅎ 일반적인 브라우니보다 밀가루 맛이 덜해서 밀도있는 초콜릿을 집어먹는 맛이 났다. 밀가루 맛을 더 좋아하긴 하지만 초콜릿 자체가 맛있어서 난 너무 맛있게 먹었구 자기는 로이스 생초콜릿 먹는것같다고 했당

                              추가로 필터커피도 시켰는데 콜롬비아 마자틀란 허니피치 무산소허니 co-fermented(복숭아, 리치, 코코넛)랑 코스타리카 볼칸아줄 카투라 내추럴(건무화과, 바닐라, 인텐스)를 골랐다. 내가 고른게 콜롬비아였는데 뚜렷한 리치/복숭아 맛과 마지막 코코넛 향이 좋아서 넘넘 맛있게 마셨다 ㅎㅎ

                              #

                              원래 브라우니는 맛있기 쉽다고 생각해서 잘 안사먹었었는데, 진짜 맛있는걸 안먹어봐서 그랬다는걸 깨달아버렸다. 서촌스코프랑 브이랩커피는 넘 맛있게 먹어서 또 먹으러 갈것같다 ㅎㅎ
