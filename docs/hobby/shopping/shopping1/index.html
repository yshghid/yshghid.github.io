                              #2025-07-16

                              연구실다니면서 맨날 오버핏반팔티만 입고다니다가,,, 출근용으로 입을게업는거같아서 급하게 맨날맨날템을 구비해보앗다

                              1-룩캐스트 케일 라운드 슬림 니트탑

                              첨에 네이비를삿는데 얇고 시원하고 막입기좋아서 깔별로 사버렸당

                              가격대비 질도 조음

                              2-룩캐스트 베린 썸머 턱 팬츠

                              매우얇고 핏이이쁘다 ㅎ

                              3-룩캐스트 코니 윈드브레이커 후디 점퍼

                              에어컨가리기용으로 그냥 사봤는데 생각보다 이뻐서 흰색도 살까말까 고민중이당

                              4-룩캐스트 무이 소프트 티셔츠

                              끔찍한 맨날맨날템.. ㅋㅋ 소재가 텐셀느낌인데 쇼핑몰텐셀티 특유의 추리해보이는느낌이 안나서 좋길래 얘도 깔별로 구매해버렷다 ㅋㅋ

                              5-라메레이 Semi Bootcut Denim

                              얘는 잘입구다니는데 사이즈를좀 잘못삿다.. ㅋㅋ S로 다시사고싶은데 사이즈 2개 똑같은거 사는거 너무 재벌행동 같아서 그냥 입음

                              6-룩캐스트 트루데이 UV 우양산

                              이건사실 사은품으로받앗는데 엄청잘쓰다가 머에걸려서 부셔졋음 근데 대체품을 못찾아서 그냥 다시들어가서 삿다 이쁘구 가벼워서 최고의양산!!

                              7-아디다스 킥 74- 코어 화이트:코어 블랙

                              연구실에서 맨날 삼선슬리퍼 신고 다니다가 깔끔템으로 하나구매함 이쁘고 때도잘안타는데 여름에신기쪼금덥다 근데 이뻐서 그냥 신음

                              8-리큐엠 보조배터리

                              먼가 출근하면 필요하지 않을까?!!해서 충동구매 했다

                              9-스카이보틀 퍼퓸핸드크림 화이트레인

                              고민영 유튜브 보다가 영업당해서 삼 ㅋㅋ 비누향나는데 생각보다 향이쎄서 많이씩못쓰고있다

                              일케 털달린 커버도 같이왔는데 여름인만큼 매우더워보이지만 ㅎㅎㅎ 기엽길래 그냥 끼워줌

                              10-게스 부츠컷데님

                              먼가 있으면 잘입지않을까?!!해서삿는데 여름보다는 가을겨울에 니트입고 입는게 이쁠거같은바지긴하다
