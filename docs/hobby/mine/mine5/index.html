                              사고싶은 화장품들 #

                              #2026-07-22

                              율리 유튜브 보다가 섀도우랑 이것저것 사고싶은게 생겼다 ㅎㅎ

                              첫번째는 홀리카홀리카의 마이페이브 무드 아이팔레트!

                              율리가 레터프롬스프링 색상 사용한거보고 예뻐서 사고싶었던건데 색상 자체는 얼모스트그레이프가 나한테 더 잘어울릴것같아서 고민이 된당

                              #

                              두번째는 클리오 뮤즈 마스터 팔레트인데

                              마찬가지 율리가 쓴 꽃가마코랄 색상이 예뻐보여서 사고싶었던건데 내가 자주쓰는색상은 수놓은핑크에 가까워서.. 멀 살지 모르겠다.

                              #

                              세번째는 홀리카홀리카 마이페이브 피스 아이섀도우랑, 필리밀리 사선라이너브러시이다.

                              예전에 홀리카홀리카 섀도우를 아이라이너 그리는용으로 산적이 있긴 한데 그때 색상도 너무 웜했고 뭔가 뭉툭한 느낌? 잘 그려진다는 느낌을 못받긴 했었다. 그게 브러쉬때문인지 그때보다 제형이 꾸덕해진건지 모르겠지만 다시 보니까 또 괜찮아 보여서 ㅎㅎ 재시도해볼예정이다.

                              색상은 코코아브륄레가 괜찮을것같다 ㅎㅎ

                              #

                              마지막은 네이밍 오버듀 글로시 립틴트!

                              색상은 나나, 듀이, 쥬시가 예뻐보여서 고민이 되는데 틴트는 발라봐야 느낌이 와서 올영을 한번 가야할것같다 ㅎ

                              #

                              요게 위 화장품들로 하고싶은 메컵 ㅎㅎ

                              첫번째 사진이 마이페이브무드 아이팔레트를 사용한 조금 화사한 느낌, 아래는 클리오 뮤즈마스터 팔레트에 네이밍 오버듀글로시 립틴트 듀이 색상을 사용한 약간 차분한 느낌의 메이크컵이당

                              첫번째 사진의 화사한데 따뜻한 느낌을 낼려면 블러셔를 웜한걸 써줘야할것같은데 율리는 시에나 컬러를 써줬다.

                              나였으면 플러터나 아이싱을 샀을것같긴한데,, 막상 올려놓은걸 보면 시에나도 예뻐보이고 해서 고민이 된당
