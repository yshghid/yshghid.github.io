                              시너진 부츠컷 데님 #

                              #2026-07-24

                              시너진은 인스타나 유튜브 광고로 엄청 많이 봤었는데 (지젤 청바지 등등) 너무 홍보 원툴 브랜드같은 느낌이 들어서 살생각은 안해봤다.

                              얼마전에 29cm 구경하다가 워싱이 엄청 예쁜 연청 데님이 있길래 가격은 좀 있지만 사고싶은데 하고 브랜드를 보니까 시너진이었다 ㅎ 안그래도 내심 궁금했는데 옷도 예쁜것같구 리뷰가 많으니까 사이즈 고르기가 수월해서 이번에 외출용 청바지로 구매해봤다 ㅎㅎ

                              색상은 라이트블루, 딥블루 이렇게 2개 구매했다!

                              사이즈는 고민끝에 m을 샀는데 원래였으면 l을 샀을것같긴하지만 l은 밑단너비가 너무 길었고 리뷰에서 사이즈업 굳이 필요없다는 말도 많아서 그냥 m으로 구매했다.

                              그리고 슬픈점이 원래는 라이트블루가 갖고싶은거였고 딥블루는 보다보니까 리뷰도 많고 색상도 빈티지하게 잘 빠진것같아서 그냥 사본거였는데 라이트블루가 신상이라서 8.24 이후 배송이었다… ㅠㅠ 빨리 입고싶은데…

                              #

                              그리고 무신사나 29cm에는 없는 색상인 그레이가 공홈에 있어서 요것도 갖고싶다 ㅎㅎ 딥블루가 맘에 들면 이것도 살것같다.

                              #

                              그리고 같이 입을 용도로 와르포우먼에서 슬림 라인 반팔 티셔츠도 구매했다 ㅎ

                              이 데님이 밑위가 23.5로 좀 짧은 편이라서 집에있는 세미크롭티들은 조금 짧을것같았다 그리고 넣어입는게 예쁠 데님도 아닌것같아서 적절한 기장의 무지 반팔티를 열시미 찾았는데 요게 가격도 저렴하고 (무려 23000원) 사이즈도 맘에 들어서 요걸로 화이트/블랙 구매해줬다.

                              사이즈는 m으로 샀구 데님 디자인상 윗버튼이 보여야 예쁠것같아서 총장 53이 딱 적당해보인다 ㅎㅎ

                              모델컷을 보면 아주 조금 아래가 둥글림 처리가 돼있는데 이것땜에 빼서 입었을때 자연스럽게 예쁠것같아서 더 맘에 들었다!

                              다른 색상으로는 그레이도 있는데 예쁘면 그레이도 살것이다 ㅎ
