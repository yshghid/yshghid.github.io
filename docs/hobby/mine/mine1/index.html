                              유어마인드 69개의 책갈피 #

                              #2026-06-22

                              인스타를 보다가 연희동의 유어마인드라는 서점에서 69개의 책갈피라는 책갈피 팝업을 한다고 해서 방문해봤다 ㅎㅎ

                              온라인에서 이미한번 보고갔는데도 온라인에는 없는 책갈피들도 많은데다 디피가 넘 예뻐서 생각보다 더더 오래 구경했다!!

                              내가 사온건 요거 두개였는데, 하나는 Insight를 얻는다는걸 전기오른걸로 표현한게 넘 귀엽고 내가 추구하는 독서여서 사고싶었고, 하나는 ‘지성의 탄생’이라는 키워드가 맘에 들고 일러스트도 예뻐서 골랐다.

                              더 사고싶은것도 많았지만 정 갖고싶으면 온라인으로 사자는 마인드로 안사고 왔는데, 막상 언박싱해보니 생각보다 더더 만족스러워서 ㅎㅎ 온라인 들어가서 2차 쇼핑을 해줬다 ㅋㅋㅎㅎ

                              #

                              첫번째는 자기가 예쁘다고 했던 MY ROOM AUDIO SYSTEM CATALOG인데

                              나는 요 2번 삽화가 맘에 들구

                              자기는 1,4번 삽화를 맘에들어했을것같다. 요건 서프라이즈로 구매해봐도 좋을것같아서 고민중이다 ㅎㅎ

                              #

                              두번째는 요 읽는중입니다(Do Not Disturb) 책갈피! 요건 온라인에서 예쁘다고 생각해서 봐두고간건데 오프라인 품절이어서 못샀다. 실물도 꽤 예뻤어서 요건 갖고싶다 ㅎㅎ

                              기욥다 ㅎㅎ

                              #

                              요건 예전에 심플책방이라는 독립서점에서 봤던것도 같은 일러스트인데 예뻐서 살까말까 고민된다.

                              앞뒷면은 요렇게 생김. 실용성은 좋은데 조금 애매하다 ㅎ

                              #

                              추가로 내가 산 Cosmic Star 책갈피가 넘 맘에들어서 선물용으로 몇개더살까 해서 봤는데 69개의 책갈피에는 없고 68개의 책갈피라는 인덱스로 유어마인드 책갈피 섹션에 올라와있었다. 아마 지난 시즌 상품인듯하다.

                              요거 역시 살까말까 고민중..!
