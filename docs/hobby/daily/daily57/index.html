                    첫월급으로 골라본 엄마아빠 선물 ㅎㅎ

                    배송 늦을까봐 일치감치 결제했는데 엄마껀 5.7, 아빠꺼는 5.8 배송 예정이다 ㅎㅎ

                    3. 그리고 다음달부터는 150만원씩 저축하는게 적당할것같다.
