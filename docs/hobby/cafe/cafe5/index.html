                까사넬로 #

                여기는 길이 데리고가줘서 알게된 카페인데 내부가 예쁘고 케이크 종류가 많았다!

                이중에 베리잼 올라가있는 치즈케이크 포장해갔는데 진짜 맛있었고 포장이 되게 예뻐서 선물용으로도 좋을듯함.

                몇달뒤에 엄마랑 다시가서 에스프레소 콘파냐 시켜봤는데 그냥 무난무난했당. 첫 에스프레소콘파냐를 너무 맛있는집에서 먹었나봐… 암튼 총평은 내부가 예쁘고 케이크가 맛있고 케이크 포장이 예쁜 집.
