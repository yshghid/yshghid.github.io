                이씨에그타르트 동성로점 #

                먹을때마다 너무 맛있어서 도파민나오는 에그타르트집.

                오리지널 콘 바닐라 누룽지 토마토바질 블루베리 먹어봤는데 오리지널이 제일 맛있고 누룽지 토마토바질이 은근 독특하고 맛있었다. 특히 토마토바질은 안어울릴거같은데 잘어울려서 말이안됨. 근데 오리지널이 제일 맛있다. 가격 저렴하거나 집에서 가까웠으면 너무 많이 사먹었을듯. 비싸고 멀어서 다행이다.
