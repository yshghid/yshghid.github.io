                프릳츠 도화점 #

                학회때 오전에 잠깐 들릴 카페 찾다가 이뻐서 가봤는데 생각보다 더더 예뻤고 커피도 넘 맛있었당 근데 카페에서 교수님 만나서 빵 사주심. ㅋㅋ

                너무 마음에 들었던 나머지 점심먹고 다같이 카페갈때 여기가자해서 또 감… ㅋㅋㅋ 커피 말고 겨울시그니처인 핫초코같은걸 시켰는데 얘도 맛있었다

                집와서 다음날 교수님이사주신 빵 먹어봄. 무난무난한 페스츄리 소세지빵맛이었다 여긴 커피가 맛있는듯.

                프릳츠 제주성산점 #

                성산일출봉 카페 찾다가 전에 기억이 좋앗던 프릳츠가 있길래 여기 가자구했다!! 예상은 했지만 사람이 진짜 많았음. 커피는 역시나 맛있다.

                바자리에 앉아있다가 창가자리 나서 옮김. 뷰 예쁜데 사실 내부 다 예뻐서 아무데나 앉아도 괜찮은 카페당
