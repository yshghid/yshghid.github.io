                ｢보헤미안 랩소디｣ #

                영화관가서 처음봤을때 넘슬펐는데 볼때마다 다른장면이 눈에들어옴 특별한영화

                ｢월플라워｣ #

                외로울때 주기적으로 시청해줘야하는 영화

                ｢노트북｣ #

                볼수록 여주가 쓰레기지만 그만큼 로맨틱함..

                ｢어바웃타임｣ #

                완벽한 로맨스 영화. 어느정도냐면 주기적으로 시청해서 로맨스영화에 대한 에임을 맞춰줘야함

                ｢비포선셋｣ #

                제목답게 선셋 씬이 제일 예쁨 / 넷플릭스에 원래 안나왔는데 이제 나와서 좋다 ㅎㅎㅎ

                ｢비포선라이즈｣ #

                대사하나하나 낭만그자체 마지막에들어간 술집이 제일 좋았당

                ｢비포미드나잇｣ #

                비교적 여주 남주사이 관계보다 그 시기의 사람들과의 토크가 더 인상깊긴함 그래도기분좋아지는영화!!

                ｢아메리칸 셰프｣ #

                가볍게 기분좋아지고싶을때 꺼내보는영화

                ｢카페소사이어티｣ #

                ｢마이크롭앤가솔린｣ #

                귀엽고 재밌는데 얘네한테 나자신 투영돼서 그날밤 좀우울해졋엇음.

                ｢노인을 위한 나라는 없다｣ #

                이거솔직히 뭘말하고싶은건지 모르겠고 그냥 범죄영환가? 생각들어서 찾아봣는데 노인을 위한 나라는 없다 (No Country For Old Men, 2007) 이거 보고 전반적으로 이해돼서 좋앗던영화로 승격 ㅎㅎ

                ｢트와일라잇｣ #

                오글,, 이지만 그맛에봣지만 그래도 오글
