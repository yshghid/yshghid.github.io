                      Genome Curation #4 Genomic Data Collection Variables

                      Genome Curation #4 Genomic Data Collection Variables #

                      #2026-04-02

                      시퀀싱 데이터의 read mapping으로 bam 파일을 만들고 variant calling으로 VCF를 만들고 VCF를 annotation한 후 분석 결과 report를 생성한다. 연구든 임상이든, 결과를 체계적이고 표준화된 형식으로 기록하지 않으면 다른 기관과 비교할 수 없고, 시간이 지나면 자기 자신조차 해석할 수 없게 된다.

                      유전체 분석 결과 report에서 반드시 수집해야 하는 변수(variable)들은 크게 네 가지 범주로 나뉜다.

                      1. 검체 및 시퀀싱 정보  ← 어떤 시료를, 어떤 방법으로 분석했는가
2. SNV 정보            ← 단일 염기 변이와 짧은 삽입/결실
3. CNV 정보            ← 복제 수 변이
4. Fusion 정보         ← 유전자 융합

                      #

                      #1 검체 및 시퀀싱 정보

                      검체 번호 (Specimen ID)

                      모든 기록의 출발점은 검체 번호다. 이것은 분석 대상이 된 생물학적 시료(조직, 혈액 등)에 부여된 고유 식별자로, 연구검체번호라고도 한다. 한 환자에게서 여러 시점, 여러 부위의 검체를 채취할 수 있으므로, 검체 번호는 환자 번호와 다르다. 예를 들어 환자 P001의 원발 종양에서 채취한 검체는 SP001-T1, 전이 부위에서 채취한 검체는 SP001-M1처럼 구분된다. 이 번호가 이후 모든 변이 기록에 연결되는 열쇠(key)가 된다.

                      #

                      검체의 질 (Quality)

                      시퀀싱 전에 추출된 DNA/RNA의 품질을 확인한 결과를 기록한다. QC(Quality Control) pass 여부를 pass, fail, not available 중 하나로 표시한다. QC 기준은 패널마다 다르다. 일반적으로 DNA의 양(최소 수십~수백 ng), 순도(OD 260/280 비율), 단편화 정도(DIN 값) 등을 확인한다. QC에 실패한 검체로 시퀀싱을 진행하면 읽기 품질이 낮아지고 변이 호출의 신뢰도가 떨어지므로, 이 정보를 반드시 기록해두어야 나중에 결과 해석 시 품질 문제를 고려할 수 있다.

                      #

                      검사 결과: 구조화된 데이터 기반 (Structured Data Based Result)

                      개별 변이 외에 유전체 전반의 특성을 나타내는 요약 지표가 있다. 대표적인 것이 종양변이부담과 반복서열불안정성이다.

                      종양변이부담(Tumor Mutational Burden, TMB) 은 종양 유전체에 축적된 체세포 돌연변이의 총량을 나타내는 지표다. 보통 코딩 영역 메가베이스(Mb)당 변이 수로 표현한다. 예를 들어 TMB가 10 mut/Mb라면 코딩 영역 100만 bp당 변이가 10개라는 뜻이다.

                      TMB가 왜 중요한가? 변이가 많은 종양은 비정상적인 단백질(신생항원, neoantigen)을 많이 만들어낸다. 면역 체계는 이 비정상 단백질을 “외부 침입자"로 인식할 수 있다. 따라서 TMB가 높은 종양은 면역관문억제제(immune checkpoint inhibitor, 예: pembrolizumab)에 더 잘 반응하는 경향이 있다. 실제로 FDA는 TMB-High(≥10 mut/Mb)인 고형암에 대해 pembrolizumab의 사용을 승인했다. 이 때문에 TMB는 단순한 통계 수치가 아니라 치료 방침을 결정하는 바이오마커(biomarker)로서 반드시 기록되어야 한다.

                      #

                      반복서열불안정성(Microsatellite Instability, MSI) 은 유전체 곳곳에 있는 반복서열(microsatellite)의 길이가 정상 조직과 종양 조직 사이에서 달라졌는지를 나타내는 지표다.

                      반복서열이란 CACACACA… 처럼 짧은 서열 단위가 여러 번 반복되는 구간이다. 이런 반복 구간은 DNA 복제 시 중합효소가 미끄러지기(slippage) 쉬워서 오류가 잘 발생한다. 정상 세포에서는 DNA 불일치 수선(mismatch repair, MMR) 시스템이 이 오류를 고쳐주지만, MMR 유전자(예: MLH1, MSH2, MSH6, PMS2)에 돌연변이가 생기면 수선이 안 되어 반복서열 길이가 변한다. 이것이 MSI다.

                      MSI는 보통 MSI-High(MSI-H, 불안정), MSI-Low(MSI-L), 또는 Microsatellite Stable(MSS, 안정)로 분류된다. MSI-H 종양은 TMB-High와 마찬가지로 면역관문억제제에 잘 반응하므로, FDA는 MSI-H인 고형암에도 pembrolizumab을 승인했다. 이것은 암의 발생 부위(폐, 대장, 위 등)가 아니라 분자적 특성으로 치료제를 선택한 최초의 사례 중 하나여서 정밀의학의 이정표로 꼽힌다.

                      #

                      패널명 및 버전 (Panel Name & Version)

                      어떤 유전자 패널을 사용했는지 기록한다. 상용 패널이라면 브랜드 이름(예: FoundationOne CDx, Oncomine, TruSight Oncology 500)을, 기관 자체 개발 패널이라면 in-house임을 명시하고 버전을 기록한다. 패널 정보에는 분석 대상 유전자의 개수와 이름 목록이 반드시 포함되어야 한다. 같은 “암 유전자 패널"이라 해도 50개 유전자를 보는 패널과 500개 유전자를 보는 패널은 결과가 다를 수밖에 없다. 특정 유전자에서 변이가 발견되지 않았을 때, 그것이 “분석했는데 없었다"인지 “분석 대상에 포함되지 않았다"인지를 구분하려면 패널에 어떤 유전자가 포함되어 있었는지를 알아야 한다. 또한 패널 버전이 업데이트되면 분석 대상 유전자가 추가되거나 제거될 수 있으므로 버전 정보도 필수적이다.

                      #

                      샘플 타입 (Sample Type)

                      분석에 사용된 시료의 종류를 tissue(조직) 또는 liquid(액체) 로 기록한다.

                      조직 시료는 수술이나 생검으로 직접 채취한 종양 조직에서 DNA를 추출한 것이다. 종양 세포의 비율(tumor purity)이 높아서 낮은 빈도의 변이도 잘 검출된다. 액체 시료는 주로 혈액에서 채취한다. 종양 세포가 파괴되면서 혈류에 방출하는 DNA 조각인 순환종양DNA(circulating tumor DNA, ctDNA) 를 분석한다. 이것을 액체생검(liquid biopsy) 이라 부른다. 비침습적이라는 장점이 있지만, 혈액 속 ctDNA의 비율이 매우 낮으므로(전체 cell-free DNA의 0.1~10% 수준) 매우 깊은 시퀀싱과 민감한 분석 기법이 필요하다. 조직과 액체에서 검출되는 변이 프로파일이 다를 수 있으므로 어떤 시료 유형을 사용했는지 반드시 기록해야 한다.

                      #

                      NGS 기법 (NGS Technique)

                      시퀀싱에 사용된 구체적 기법을 기록한다. 각 기법은 라이브러리 제작 방식, 분석 범위, 적용 목적이 다르다.

                      PCR-amplicon (표적 유전자 패널) 은 관심 있는 유전자 영역을 PCR로 직접 증폭하는 방법이다. 특정 위치에 대한 프라이머를 설계하여 해당 부분만 선택적으로 증폭한다. 비용이 적고 빠르지만, 프라이머가 결합하지 못하는 영역은 분석할 수 없고, 균일하지 않은 증폭(amplification bias)이 발생할 수 있다.

                      Hybrid-capture enrichment (표적 유전자 패널) 은 관심 영역에 상보적인 짧은 DNA/RNA 탐침(probe)을 사용하여 해당 영역의 DNA 조각만 낚아내는(capture) 방법이다. PCR-amplicon보다 균일한 커버리지를 얻을 수 있고, 더 넓은 영역을 분석할 수 있다. 현재 대부분의 상용 암 유전자 패널이 이 방식을 사용한다.

                      RNA sequencing 은 DNA가 아닌 RNA를 읽는 것이다. 유전자 발현량 분석, 융합 유전자 검출, 스플라이싱 변이 탐지에 사용된다. 특히 DNA에서는 찾기 어려운 융합 유전자를 RNA에서 더 쉽게 검출할 수 있는데, 전좌로 인해 만들어진 융합 mRNA가 직접 읽히기 때문이다.

                      DNA based (WES and WGS) 는 엑솜 전체(WES) 또는 유전체 전체(WGS)를 읽는 방법이다. 패널보다 분석 범위가 넓어 새로운 변이를 발견할 가능성이 높지만, 비용이 많이 들고 데이터 분석에 시간이 오래 걸린다. 기타(other) 에는 메틸화 분석, long-read 시퀀싱, single-cell 시퀀싱 등 위에 해당하지 않는 기법이 포함된다.

                      #

                      #2 SNV 정보

                      SNV(Single Nucleotide Variant)와 짧은 삽입/결실(Indel)을 포함하는 변이 정보를 기록하는 항목이다. 각 변이마다 다음 변수들을 수집한다.

                      #

                      검체 번호 (Specimen ID)

                      이 변이가 어떤 검체에서 발견되었는지를 연결하는 식별자다. 앞서 검체 및 시퀀싱 정보에서 기록한 검체 번호와 동일한 값을 사용한다.

                      #

                      변이 정보 - SNV

                      유전자(Gene) 는 변이가 위치한 유전자의 이름이다. 예를 들어 EGFR, BRAF, TP53 등이다. Variant calling 후 ANNOVAR나 VEP 같은 annotation 도구가 좌표를 유전자 이름으로 변환해준 결과를 기록한다.

                      염기서열 변화(DNA-Change) 는 DNA 수준에서 어떤 변화가 일어났는지를 HGVS(Human Genome Variation Society) 표기법으로 기록한 것이다. 예를 들어 c.1799T>A는 코딩 서열의 1799번째 염기가 T에서 A로 바뀌었다는 뜻이다. 여기서 c.는 코딩 DNA 서열 기준(coding DNA reference sequence)이라는 접두사다.

                      아미노산 서열 변화(Amino Acid Change) 는 단백질 수준에서 어떤 변화가 일어났는지를 기록한 것이다. 예를 들어 p.V600E는 단백질의 600번째 아미노산이 발린(V, Valine)에서 글루탐산(E, Glutamic acid)으로 바뀌었다는 뜻이다. p.는 단백질 서열 기준(protein reference sequence)이라는 접두사다. BRAF p.V600E는 흑색종에서 가장 흔한 드라이버 돌연변이로, 베무라페닙(vemurafenib) 같은 표적 치료제의 적응증이 된다. DNA-Change와 Amino Acid Change를 함께 기록하는 이유는, 같은 아미노산 변화가 여러 가지 DNA 변화에서 올 수 있기 때문이다(유전암호의 중복성 때문).

                      Read depth 는 해당 변이 위치를 덮는 읽기의 총 수다. VCF 파일의 DP 필드에 해당한다. read depth가 높을수록 변이 호출의 통계적 신뢰도가 높다. 예를 들어 read depth가 500이라면 해당 위치를 500번 읽었다는 뜻이고, read depth가 10이라면 겨우 10번 읽은 것이므로 통계적 판단이 불안정하다. 일반적으로 패널 시퀀싱에서는 수백~수천의 read depth를 목표로 한다.

                      VAF(%)(Variant Allele Frequency, 변이 대립유전자 빈도) 는 해당 위치를 덮는 전체 읽기 중 변이 염기를 가진 읽기의 비율을 백분율로 나타낸 것이다. VCF 파일의 AF 필드에 해당한다.

                      VAF의 의미를 직관적으로 이해하면 이렇다. read depth가 200이고 그중 100개가 변이 염기를 가지고 있다면 VAF는 50%다. 정상 이형접합 생식세포 변이(germline heterozygous)라면 VAF는 이론적으로 50%이고, 동형접합(homozygous)이라면 100%다. 체세포 돌연변이의 경우 종양 순도(tumor purity)와 종양 내 아클론 구조(clonality)에 따라 VAF가 다양하게 나타난다. VAF가 5%라면 시료 속 대부분의 세포는 정상이고 소수의 세포에서만 이 변이가 있다는 뜻이다. 낮은 VAF의 변이를 신뢰하려면 높은 read depth가 필수적이다.

                      변이 유형(Variant Type) 은 이 변이가 단백질에 어떤 영향을 미치는지를 분류한 것이다. 8가지 범주로 나뉜다.

                      nonsense는 코돈이 정지 코돈으로 바뀌어 단백질이 중간에서 잘리는 것이다. missense는 하나의 아미노산이 다른 아미노산으로 바뀌는 것이다. frameshift는 3의 배수가 아닌 삽입/결실로 읽기 틀이 어긋나는 것이다. inframe은 3의 배수만큼의 삽입/결실로 읽기 틀은 유지되면서 아미노산이 추가되거나 빠지는 것이다. splicing은 인트론-엑손 경계의 스플라이싱 접합부에 발생한 변이다. silent는 코돈이 바뀌었지만 같은 아미노산을 지정하는 경우다. others와 unknown은 위 범주에 해당하지 않거나 분류가 불확실한 경우다.

                      #

                      변이의 임상적 분류 (Clinical Significance / Tier)

                      발견된 변이가 실제로 질병과 관련이 있는지를 판정하는 분류 체계다. 이것은 단순히 변이의 종류(missense, nonsense 등)를 넘어서, 기존 문헌, 데이터베이스, 기능 연구, 인구 집단 빈도 등을 종합적으로 고려한 해석(interpretation) 의 결과다.

                      Pathogenic (Tier 1, 병원성) 은 이 변이가 질병을 일으킨다는 강력한 증거가 있는 경우다. 다수의 독립적인 연구에서 질병과의 인과관계가 확인되었고, 기능 연구에서 단백질 기능에 해로운 영향이 입증되었으며, ClinVar나 COSMIC 같은 데이터베이스에 병원성으로 등록되어 있다. 임상적으로 가장 중요한 변이이며, 치료 방침에 직접적인 영향을 준다.

                      Likely pathogenic (Tier 2, 병원성 가능) 은 병원성일 가능성이 높지만, Tier 1만큼 완전한 증거가 모이지는 않은 경우다. 예를 들어 같은 유전자의 다른 위치에서 이미 병원성으로 확인된 변이와 유사한 패턴을 보이지만, 이 특정 변이 자체에 대한 직접적인 기능 연구가 아직 없는 경우에 해당한다. 임상적으로는 pathogenic과 비슷하게 취급하되, 추가 연구가 필요하다는 뉘앙스가 있다.

                      Uncertain significance (Tier 3, VUS, 의의 불명 변이) 은 병원성인지 양성인지 판단할 수 없는 경우다. 데이터가 부족하거나 증거가 상충하는 경우에 해당한다. 임상에서 VUS는 가장 다루기 어려운 결과인데, 환자에게 “변이가 있지만 의미를 모르겠다"고 전달해야 하기 때문이다. 시간이 지나면서 추가 연구가 축적되면 pathogenic이나 benign으로 재분류될 수 있다.

                      Benign (Tier 4, 양성) 과 Likely benign (Tier 4, 양성 가능) 은 이 변이가 질병과 관련이 없다고 판단된 경우다. 일반 인구 집단에서 흔하게 관찰되거나(gnomAD 등에서 높은 빈도), 기능 연구에서 영향이 없음이 확인된 경우에 해당한다.

                      Others와 Unknown은 위 분류 체계에 맞지 않거나 아예 분류 시도가 이루어지지 않은 경우다.

                      #

                      #3 CNV 정보

                      복제 수 변이(Copy Number Variation)를 기록하는 항목이다. SNV와 달리 단일 염기가 아니라 DNA의 넓은 구간이 통째로 증가하거나 감소한 것이므로, 기록하는 변수의 성격이 다르다.

                      #

                      검체 번호 (Specimen ID)

                      SNV와 동일하게 이 CNV가 발견된 검체를 식별하는 번호다.

                      #

                      변이 정보 - CNV

                      유전자(Gene) 는 복제 수 변이의 영향을 받는 유전자 이름이다. 하나의 CNV 구간 안에 여러 유전자가 포함될 수 있다.

                      복제수 변이 부위(Locus) 는 변이가 발생한 유전체 상의 구간을 나타내는 좌표 정보다. 넓은 의미의 위치를 기술하며, 구체적인 좌표는 아래의 chromosome, start position, end position으로 정확히 지정된다.

                      Chromosome, Start position, End position 은 CNV 구간의 정확한 게놈 좌표다. 예를 들어 chromosome=7, start=55086714, end=55324313이면 7번 염색체의 55,086,714번째부터 55,324,313번째 위치까지 약 237kb 구간에서 복제 수 변이가 발생했다는 뜻이다. 이 구간이 EGFR 유전자를 포함한다면 EGFR 증폭(amplification)으로 보고된다.

                      Copy number alteration type (gain/loss) 는 복제 수가 정상보다 증가했는지(gain) 감소했는지(loss)를 기록한다. gain은 해당 구간의 DNA가 정상보다 많이 복제된 것이고, loss는 일부 또는 전부가 소실된 것이다. 암에서 종양유전자(oncogene)의 gain은 단백질 과발현으로 이어져 세포 성장을 촉진하고, 종양억제유전자(tumor suppressor gene)의 loss는 세포 성장 억제 기능의 소실로 이어진다.

                      Fold change 는 정상 대비 읽기 깊이(read depth)의 비율이다. 정상 디플로이드(2 copy) 상태를 기준으로 fold change가 2.0이면 복제 수가 2배(4 copy)로 늘었다는 의미이고, 0.5이면 절반(1 copy)으로 줄었다는 의미다. 다만 종양 순도, 이형 접합도, 전체 배수성(ploidy)에 따라 보정이 필요하므로 fold change만으로 정확한 복제 수를 알 수는 없다.

                      Estimated copy number (CN) 는 종양 순도와 배수성을 보정한 후 추정된 실제 복제 수다. 정상은 CN=2이고, CN=0이면 동형접합 결실(homozygous deletion), CN=1이면 반접합 결실(hemizygous deletion), CN=5 이상이면 고도 증폭(high-level amplification)으로 해석한다. ABSOLUTE, FACETS 같은 알고리즘이 이 추정을 수행한다.

                      Tier 는 SNV와 마찬가지로 이 CNV의 임상적 중요도를 나타내는 분류다. ERBB2(HER2) 유전자의 증폭은 유방암에서 trastuzumab 치료의 적응증이 되므로 Tier 1에 해당하는 반면, 임상적 의미가 확인되지 않은 유전자의 복제 수 변이는 Tier 3 이하로 분류될 수 있다.

                      #

                      #4 Fusion 정보

                      유전자 융합(Gene Fusion)을 기록하는 항목이다. 전좌(translocation)나 역위(inversion), 큰 결실 등 구조 변이의 결과로 두 개의 별개 유전자가 하나로 합쳐진 경우를 기록한다.

                      #

                      검체 번호 (Specimen ID)

                      이 Fusion이 발견된 검체의 식별 번호다.

                      #

                      변이 정보 - Fusion

                      Fusion gene 는 융합 유전자의 이름을 표기한 것이다. 관례적으로 5’ 쪽 유전자(앞쪽)와 3’ 쪽 유전자(뒤쪽)를 하이픈으로 연결해서 표기한다. 예를 들어 EML4-ALK는 EML4 유전자의 앞쪽 부분과 ALK 유전자의 뒤쪽 부분이 합쳐진 융합 유전자다.

                      5’과 3’의 구분이 중요한 이유가 있다. 대부분의 융합 유전자에서 5’ 파트너 유전자는 프로모터(유전자 발현을 시작시키는 스위치)와 앞쪽 엑손을 제공하고, 3’ 파트너 유전자는 기능적 도메인(예: 키나제 도메인)을 제공한다. EML4-ALK의 경우, EML4의 프로모터가 ALK의 키나제 도메인을 끊임없이 발현시켜서 세포 성장 신호가 꺼지지 않게 만든다. 이것이 비소세포폐암의 약 3~5%에서 발견되며, ALK 억제제(crizotinib, alectinib 등)의 치료 표적이 된다. Gene1 과 Gene2 는 융합에 참여한 각각의 유전자 이름이다. 위 예에서 Gene1은 EML4, Gene2는 ALK다.

                      Gene1 breakpoint 와 Gene2 breakpoint 는 각 유전자에서 절단이 일어난 정확한 게놈 좌표다. 예를 들어 Gene1 breakpoint가 chr2:42522656이고 Gene2 breakpoint가 chr2:29446394라면, 2번 염색체 위의 이 두 위치에서 DNA가 끊어져서 재결합되었다는 뜻이다. EML4와 ALK는 모두 2번 염색체에 있지만 원래 멀리 떨어져 있다가, 역위(inversion)에 의해 합쳐진다.

                      breakpoint의 정확한 좌표가 왜 중요한가? 같은 EML4-ALK 융합이라도 breakpoint 위치에 따라 여러 변이체(variant) 가 존재한다. EML4의 어느 엑손까지 포함되느냐에 따라 variant 1, 2, 3 등으로 나뉘고, 각 variant가 ALK 억제제에 대한 반응이 다를 수 있다는 연구 결과가 있다. 따라서 단순히 “EML4-ALK 양성"이라고만 기록하면 정보가 불충분하고, 정확한 breakpoint를 기록해야 한다.

                      Fusion supporting reads 수 는 이 융합 이벤트를 지지하는 읽기의 개수다. SNV의 read depth와 유사한 개념으로, 융합 접합부(fusion junction)를 가로지르는 읽기가 몇 개인지를 기록한다. supporting reads가 많을수록 이 fusion call의 신뢰도가 높다. supporting reads가 2~3개에 불과하다면 거짓 양성의 가능성을 신중히 고려해야 한다.

                      검출방법(tested with DNA or RNA) 는 이 융합 유전자를 DNA 기반 분석으로 찾았는지 RNA 기반 분석으로 찾았는지를 기록한다. DNA에서는 breakpoint 주변의 구조 변이로 탐지하고, RNA에서는 융합 mRNA가 직접 읽힌다. RNA 기반 검출은 실제로 전사(transcription)되어 발현되는 융합만 찾아내므로 기능적으로 의미 있는 융합을 잡는 데 유리하다. 반면 DNA 기반 검출은 전사 여부와 무관하게 구조적 변이 자체를 찾으므로, RNA가 발현되지 않는 융합도 탐지할 수 있다. 두 방법의 감도와 특이도가 다르므로 어떤 방법으로 검출했는지는 결과 해석에 영향을 준다.

                      Tier 는 이 융합 유전자의 임상적 중요도 분류다. EML4-ALK, BCR-ABL, ROS1 융합 등 FDA 승인 표적 치료제가 있는 융합은 Tier 1으로, 임상시험 중인 표적이 있는 융합은 Tier 2로 분류되는 식이다.

                      #

                      네 범주의 관계: 전체 그림

                      모든 변이 기록은 Specimen ID를 통해 검체 및 시퀀싱 정보와 연결된다. 이 구조 덕분에 “이 변이가 어떤 환자의, 어떤 시료에서, 어떤 패널로, 어떤 기법으로 분석된 결과인지"를 추적할 수 있다.

                      #

                      #5 핵심 요약

                      • 검체 및 시퀀싱 정보는 모든 분석 결과의 맥락을 제공한다. Specimen ID, QC 결과, TMB/MSI 같은 요약 지표, 패널명과 버전, 시료 유형, NGS 기법이 반드시 기록되어야 한다.
                      • SNV 정보는 유전자명, DNA/단백질 수준의 서열 변화, read depth, VAF, 변이 유형(nonsense, missense 등), 임상적 분류(Tier 1~4)를 포함한다. VAF와 read depth의 조합이 변이 호출의 신뢰도를 결정한다.
                      • CNV 정보는 정확한 게놈 좌표(chromosome, start, end), gain/loss 구분, fold change, 추정 복제 수(CN), Tier를 포함한다.
                      • Fusion 정보는 융합 유전자 이름, 참여 유전자와 각각의 breakpoint 좌표, supporting reads 수, 검출 방법(DNA/RNA), Tier를 포함한다.
                      • 모든 변이 기록은 Specimen ID로 검체 정보에 연결되며, 이 연결 구조가 있어야 분석 결과의 추적, 비교, 재현이 가능하다.
