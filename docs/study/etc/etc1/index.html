              SK AX 인성검사 준비 #

              #2025-06-19

              1. 서류 전형 #

              서류가 적부심사였는지.. 붙어버려서 생애첫 인성검사 준비를 하게되었다

              2. SKCT 인성검사 #

              SK 인재상은 패기, 실행력, 협업, 윤리의식이구

              • 이 블로그 글을 보니깐 솔직하게 하되 인재상에 맞추는것두 어느정도 필요하다는거같아서
              • 잡플래닛에서 유료 검사랑 이런저런 무료 검사도 해보겟지만
                • 일단 챗지피티에 돌려서 예시문제 100개 뽑음.
                  • 노션에 문제만 붙여넣기 해서 좀 외운담에 실제 인성검사연습을 해볼려고한다.

              SKCT 인성검사 예시 문항 120개 #

              1. 나는 변화를 두려워하기보다는 새로운 시도를 즐긴다. → 매우 그렇다 / 패기를 중요시하는 SK는 변화를 받아들이고 도전하는 태도를 높게 평가합니다.
              2. 실패하더라도 도전해보는 것이 가치 있다고 생각한다. → 매우 그렇다 / 결과보다는 도전 정신과 실행을 중시합니다.
              3. 지시받은 일 외에 스스로 추가로 할 일을 찾으려 한다. → 매우 그렇다 / 자기주도성과 적극성은 SK가 추구하는 실행형 인재상입니다.
              4. 새로운 기술이나 트렌드에 대한 공부는 시간 낭비라고 생각한다. → 전혀 아니다 / 변화에 대응하려는 학습 의지는 패기의 핵심입니다.
              5. 조직의 목표를 위해 내 역할을 자율적으로 조정할 수 있다. → 매우 그렇다 / 유연하게 역할을 재정의할 수 있는 태도는 실행력을 반영합니다.
              6. 다른 사람과 협업하는 것보다 혼자 일하는 것이 더 편하다. → 전혀 아니다 / 협업과 소통을 중시하는 SK 조직문화와 맞지 않습니다.
              7. 내가 옳다고 생각하는 일이라면 팀의 의견과 달라도 밀어붙인다. → 전혀 아니다 / 공동의 의견을 존중하고 조율할 수 있어야 합니다.
              8. 팀의 성과보다는 개인의 성과가 더 중요하다고 생각한다. → 전혀 아니다 / 공동체 기여와 협력적 성과 달성이 더 중요하게 평가됩니다.
              9. 어려운 일이 주어지면 부담스럽고 피하고 싶다. → 전혀 아니다 / 도전을 회피하지 않는 자세가 필요합니다.
              10. 팀원이 실수를 했을 때 먼저 도와주려고 한다. → 매우 그렇다 / 동료에 대한 배려와 협업 태도는 중요한 역량입니다.
              11. 실수를 숨기는 것이 팀 전체에 더 나을 때도 있다고 생각한다. → 전혀 아니다 / 투명성과 윤리성은 SK가 중시하는 가치입니다.
              12. 프로젝트의 마감 기한을 지키기 위해 다른 업무를 미루는 경우도 있다. → 매우 그렇다 / 우선순위 판단과 실행력의 반영입니다.
              13. 내가 맡은 일은 끝까지 책임지고 마무리한다. → 매우 그렇다 / 실행 중심의 결과지향적 태도입니다.
              14. 동료가 어려움을 겪고 있을 때 먼저 손을 내미는 편이다. → 매우 그렇다 / 협업과 공동체 의식을 중요시하는 SK 문화에 부합합니다.
              15. 실적이 좋지 않으면 변명을 하거나 상황 탓을 하게 된다. → 전혀 아니다 / 자기 책임을 회피하지 않는 태도가 필요합니다.
              16. 다른 팀의 성과에 대해서도 관심을 가지고 응원한다. → 매우 그렇다 / 전체 조직 성과에 대한 관심과 배려를 나타냅니다.
              17. 나는 항상 정직하게 행동하려고 노력한다. → 매우 그렇다 / 윤리성과 신뢰는 모든 판단의 기준입니다.
              18. 규칙이 비효율적이라고 느껴지면 무시할 때도 있다. → 전혀 아니다 / 절차와 규정 준수는 윤리의식의 기반입니다.
              19. 나의 아이디어로 팀이 성과를 내면 더 큰 동기부여를 느낀다. → 매우 그렇다 / 팀 기여에 대한 주인의식은 실행력을 나타냅니다.
              20. 한 번 결심한 일은 어떤 상황에서도 포기하지 않는다. → 매우 그렇다 / 끈기 있는 실행력과 책임감의 표현입니다.
              21. 낯선 분야라도 맡겨진 일이라면 배워서라도 해내려 한다. → 매우 그렇다 / 도전정신과 학습 태도가 결합된 자세입니다.
              22. 변화를 시도하는 것보다는 현재 상태를 유지하는 것이 낫다고 생각한다. → 전혀 아니다 / 변화 수용은 패기 있는 인재의 핵심입니다.
              23. 실패한 경험이 있다면 다시는 그 시도를 하지 않으려 한다. → 전혀 아니다 / 실패를 통해 배우고 재도전하는 자세가 필요합니다.
              24. 정해진 업무 외에 새로운 기회를 찾아 행동에 옮긴 적이 있다. → 매우 그렇다 / 자기주도적 문제해결력의 반영입니다.
              25. 내가 부족하다고 느끼는 부분은 적극적으로 보완하려 한다. → 매우 그렇다 / 성장 마인드셋은 패기의 핵심 요소입니다.
              26. 일상적인 반복 업무는 내게 큰 의미가 없다고 느낀다. → 전혀 그렇지 않다 / 기본 업무에 대한 성실성도 중요합니다.
              27. 나는 어떤 환경에서도 도전적인 태도를 유지하려 노력한다. → 매우 그렇다 / 일관된 태도와 패기 있는 마음가짐을 나타냅니다.
              28. 내가 주도적으로 기획하거나 제안한 프로젝트가 있다. → 매우 그렇다 / 기획과 주도적 태도는 실행력의 출발점입니다.
              29. 나의 실수나 실패를 솔직히 공유하고자 한다. → 매우 그렇다 / 정직성과 팀 신뢰 구축의 핵심입니다.
              30. 지금의 성과에 안주하기보다 다음 성장을 준비한다. → 매우 그렇다 / 지속적인 도전 의식이 중요한 태도입니다.
              31. 일의 성과는 노력보다 운에 따라 결정되는 경우가 많다. → 전혀 아니다 / 결과를 운이 아닌 노력과 실행으로 만들어가는 태도를 강조합니다.
              32. 나는 어려운 과제도 끝까지 해내야 한다는 책임감을 가진다. → 매우 그렇다 / 실행과 책임감은 SK의 핵심 인재 특성입니다.
              33. 일을 시작할 때 전체 일정을 스스로 계획하고 조율한다. → 매우 그렇다 / 주도적인 실행력이 요구됩니다.
              34. 결과가 좋지 않으면 나보다 타인의 책임을 먼저 생각하게 된다. → 전혀 아니다 / 자기 책임을 우선하는 태도가 필요합니다.
              35. 일을 미루는 편이며 마감이 닥쳐야 집중하는 편이다. → 전혀 아니다 / 시간관리와 자기조절 능력도 실행력의 일부입니다.
              36. 나는 다양한 방식으로 문제 해결을 시도한다. → 매우 그렇다 / 유연한 실행과 창의성은 변화 대응에 중요합니다.
              37. 단기 결과보다는 장기적 관점에서 일하는 것을 선호한다. → 매우 그렇다 / 전략적 실행과 지속 가능성 중심 태도를 반영합니다.
              38. 팀 프로젝트에서는 주로 중심 역할을 맡는다. → 매우 그렇다 / 주도성과 책임감이 반영된 문항입니다.
              39. 내가 맡은 일이 아니더라도 전체 프로젝트에 도움이 되면 돕는다. → 매우 그렇다 / 실행력과 협업을 동시에 반영합니다.
              40. 시행착오가 있더라도 빠르게 실행하는 것이 중요하다고 생각한다. → 매우 그렇다 / 완벽보다 실행 중심의 태도를 중시합니다.
              41. 나는 동료의 이야기를 끝까지 듣고 이해하려고 한다. → 매우 그렇다 / 소통과 존중은 협업 역량의 기본입니다.
              42. 팀원 간 갈등이 생기면 먼저 중재하려고 한다. → 매우 그렇다 / 협업 분위기 조성 능력은 SK에서 높게 평가됩니다.
              43. 동료와의 경쟁보다는 협력이 더 중요하다고 생각한다. → 매우 그렇다 / 경쟁보다 공동 성과 창출을 중시하는 문화입니다.
              44. 회의나 모임에서 내 의견보다 타인의 의견을 경청하는 편이다. → 매우 그렇다 / 경청은 건강한 협업의 핵심입니다.
              45. 동료의 성공에 진심으로 기뻐하는 편이다. → 매우 그렇다 / 조직 내 긍정적 관계 형성에 필수적인 태도입니다.
              46. 나는 팀 내에서 조율자 또는 연결자 역할을 자주 한다. → 매우 그렇다 / SK는 소통·조율 능력을 중요하게 평가합니다.
              47. 협업보다 혼자 처리하는 것이 시간 낭비가 적다고 생각한다. → 전혀 아니다 / 협업의 시너지를 중시하는 SK 문화와 반대됩니다.
              48. 팀의 결정과 다르더라도 내 생각을 끝까지 고집한다. → 전혀 아니다 / 융합과 조율이 가능한 유연한 협업 자세가 필요합니다.

              50. 나는 동료들이 나를 신뢰할 수 있도록 노력한다. → 매우 그렇다 / 신뢰 형성은 협업 기반을 만드는 중요한 역량입니다.
              51. 나는 항상 정직하게 행동하려고 노력한다. → 매우 그렇다 / 윤리성과 진정성은 SK가 가장 중시하는 가치입니다.
              52. 거짓말을 하더라도 좋은 결과를 낼 수 있다면 괜찮다고 생각한다. → 전혀 아니다 / 결과보다 과정의 정당성을 중시합니다.
              53. 나는 회사의 규정이나 윤리 규범을 잘 지키는 편이다. → 매우 그렇다 / 준법성과 조직 규율에 대한 태도를 반영합니다.
              54. 실수를 했을 때 책임을 회피하기보다는 인정하려 한다. → 매우 그렇다 / 윤리성과 책임감이 결합된 태도입니다.
              55. 문제가 생겼을 때 숨기기보다는 상사나 동료에게 먼저 알린다. → 매우 그렇다 / 투명한 보고와 소통은 신뢰의 바탕입니다.
              56. 내가 맡은 자료나 보고서는 항상 정확성과 진실성을 점검한다. → 매우 그렇다 / 신뢰 기반의 업무 수행 태도입니다.
              57. 정당하지 않은 지시를 받으면 바로 따르기보다는 다시 확인한다. → 매우 그렇다 / 윤리적 판단력과 용기를 반영한 문항입니다.
              58. 나는 공적인 일과 사적인 일을 명확히 구분하려 노력한다. → 매우 그렇다 / 공사 구분은 윤리적 행동의 기본입니다.
              59. 결과가 좋더라도 부정한 방법은 사용하지 않으려 한다. → 매우 그렇다 / 수단이 아닌 정당한 과정을 중시합니다.
              60. 동료의 부적절한 행동을 목격하면 침묵하기보다는 알린다. → 매우 그렇다 / 윤리의식과 용기를 동시에 요구합니다.
              61. 구성원 모두가 존중받는 문화를 만들고 싶다. → 매우 그렇다 / 배려와 존중은 SK의 행복한 조직 문화를 반영합니다.
              62. 업무 중 잘못된 것을 발견하면 바로 수정하려 노력한다. → 매우 그렇다 / 실행과 윤리의 결합된 자세입니다.
              63. 불필요한 업무도 요청받으면 정중히 수용하는 편이다. → 매우 그렇다 / 협업과 조직 기여에 대한 열린 태도입니다.
              64. 남들이 기피하는 일을 맡는 것을 꺼려하지 않는다. → 매우 그렇다 / 책임감과 팀워크를 중시하는 자세입니다.
              65. 협업보다 혼자 처리하는 것이 빠르고 정확하다고 믿는다. → 전혀 아니다 / 협업 효율을 인정하고 추구해야 합니다.
              66. 나는 직장 내 다양한 세대, 배경과 함께 일할 수 있다. → 매우 그렇다 / 포용성과 협업 다양성 수용 역량이 중요합니다.
              67. 회사의 명예를 훼손할 수 있는 행동은 조심하려 한다. → 매우 그렇다 / 조직 구성원으로서의 윤리적 책임 의식입니다.
              68. 업무상 알게 된 정보를 외부에 유출하지 않는다. → 매우 그렇다 / 정보 보안과 신뢰 유지는 매우 중요한 윤리 요소입니다.
              69. 자신보다 조직을 먼저 생각하고 행동하는 것이 중요하다고 생각한다. → 매우 그렇다 / 공동체 중심 가치관을 반영합니다.
              70. 상사의 부당한 지시가 있다면 참고 넘어가는 편이다. → 전혀 아니다 / 윤리 기준에 따라 정중히 제기하는 태도가 필요합니다.
              71. 팀원 중 실수를 반복하는 사람에게 직접 피드백을 준다. → 매우 그렇다 / 신뢰 기반의 피드백 문화에 기여합니다.
              72. 내가 맡은 일은 누가 보지 않아도 성실히 수행한다. → 매우 그렇다 / 자기 관리 및 신뢰 기반 행동입니다.
              73. 업무를 하며 타인의 공을 가로채는 행동은 옳지 않다고 생각한다. → 매우 그렇다 / 정직성과 공동체 윤리를 반영합니다.
              74. 약속한 일정을 지키는 것은 신뢰의 기본이라고 생각한다. → 매우 그렇다 / 기본적인 실행 및 윤리적 책임 태도입니다.
              75. 잘못된 일이라도 상사에게 인정받는다면 괜찮다고 느낀다. → 전혀 아니다 / 윤리와 성과는 분리해서 바라봐야 합니다.
              76. 업무상 실수를 숨긴 적이 있다. → 전혀 아니다 / 실수에 대한 투명한 인정과 대응이 중요합니다.
              77. 업무 중 개인적인 메시지를 자주 확인하는 편이다. → 전혀 아니다 / 집중력과 업무 태도 모두 SK가 평가하는 기준입니다.
              78. 나의 감정 상태가 업무에 영향을 주는 경우가 많다. → 전혀 아니다 / 감정 조절과 자기통제력은 중요한 직무 역량입니다.
              79. 복잡하고 민감한 업무를 맡아도 끝까지 해결할 자신이 있다. → 매우 그렇다 / 책임감 있고 강인한 실행력이 필요합니다.
              80. 나는 업무와 관련된 윤리 교육이나 지침을 자주 확인하려 한다. → 매우 그렇다 / 윤리의식과 자기 관리 태도를 반영합니다.
              81. 나는 실패한 일이라도 원인을 분석해 다시 도전한다. → 매우 그렇다 / 실패를 학습의 기회로 삼는 태도는 패기와 실행력을 동시에 반영합니다.
              82. 지시가 명확하지 않아도 스스로 판단해 일을 시작한다. → 매우 그렇다 / 자율적이고 주도적인 실행 태도를 중시합니다.
              83. 일의 결과가 내 기대보다 못하면 스스로에게 실망한다. → 매우 그렇다 / 높은 책임감과 자기평가 기준을 보여줍니다.
              84. 실행보다 완벽한 계획이 우선이라고 생각한다. → 전혀 아니다 / SK는 실행 중심의 태도를 더 높게 평가합니다.
              85. 새로운 문제에 직면하면 즐거운 도전으로 느낀다. → 매우 그렇다 / 도전을 긍정적으로 인식하는 태도가 중요합니다.
              86. 회사의 결정이 나의 가치관과 다르다면 무조건 따를 수 없다. → 전혀 아니다 / 개인의 판단보다 조직 조화를 우선해야 합니다.
              87. 나는 한 번 맡은 일은 스스로 끝까지 책임지고 처리하려 한다. → 매우 그렇다 / 책임감 있는 실행력이 핵심입니다.
              88. 누군가 업무를 미루면 내가 먼저 나서는 편이다. → 매우 그렇다 / 책임감과 팀워크를 동시에 반영한 태도입니다.
              89. 혼자 결정하기 어려운 일은 동료들과 논의한다. → 매우 그렇다 / 협업과 집단 의사결정 태도를 중시합니다.
              90. 팀 성과를 위해 내 의견을 양보한 적이 있다. → 매우 그렇다 / 조율력과 협업 마인드를 나타냅니다.
              91. 다른 부서와 협업할 때에도 적극적으로 소통하려 한다. → 매우 그렇다 / 유연한 커뮤니케이션 태도를 요구합니다.
              92. 협업 중 문제가 생기면 서로 책임을 나누기보다 해결책부터 찾는다. → 매우 그렇다 / 문제 해결 중심의 협업 자세입니다.
              93. 업무 중 발생한 문제가 내 책임이 아니더라도 해결에 앞장선다. → 매우 그렇다 / 공동의 문제를 함께 해결하려는 자세가 중요합니다.
              94. 타 부서에서 요청한 일이라도 정중히 대응한다. → 매우 그렇다 / 전체 조직의 조화를 중시하는 태도입니다.
              95. 불합리한 업무 분장이라 느껴져도 일단 수용하고 협의한다. → 매우 그렇다 / 유연성과 협업 기반의 문제 접근입니다.
              96. 다른 부서와의 갈등이 생기면 감정보다 원인 파악을 우선한다. → 매우 그렇다 / 감정보다 문제 해결 중심 사고가 중요합니다.
              97. 동료가 실수했을 때 그를 비난하기보다 함께 개선점을 찾는다. → 매우 그렇다 / 포용적이고 협력적인 태도가 중요합니다.
              98. 팀원 간 공로 분배가 공정하지 않아도 공동 목표를 먼저 생각한다. → 매우 그렇다 / 공동 성과와 조직 기여 우선 사고입니다.
              99. 구성원의 다양한 의견을 통합하는 역할을 자주 맡는다. → 매우 그렇다 / 소통과 조율 능력은 협업의 핵심 역량입니다.
              100. 나는 팀이 결정한 방향이라면 개인 의견과 달라도 따를 수 있다. → 매우 그렇다 / 조직 일원으로서의 책임감과 융합성 반영입니다.
              101. 팀워크보다 개인 경쟁이 더 동기부여가 된다고 느낀다. → 전혀 아니다 / 경쟁보다 협업과 공동 목표 달성이 더 중시됩니다.
              102. 사소한 규정이라도 철저히 지키려고 노력한다. → 매우 그렇다 / 사소한 규칙 준수도 윤리의식의 일환입니다.
              103. 상사의 부정확한 지시가 있을 경우 직접 확인을 요청한 적이 있다. → 매우 그렇다 / 소신 있고 윤리적인 실행 자세입니다.
              104. 나는 정해진 절차보다 빠른 결과가 더 중요하다고 생각한다. → 전혀 아니다 / 정당한 과정 없이 결과만 중시하는 태도는 위험합니다.
              105. 규정이나 절차가 불합리하면 개선 제안을 시도한 적이 있다. → 매우 그렇다 / 윤리성과 실행력이 결합된 태도입니다.
              106. 주변의 기대가 높을수록 정직하게 행동하려 노력한다. → 매우 그렇다 / 외부 시선보다 내면의 윤리의식을 강조합니다.
              107. 직장 내 윤리 문제가 발생하면 적극적으로 의견을 제시한다. → 매우 그렇다 / 침묵보다는 개선을 위한 참여가 중요합니다.
              108. 누군가 규정을 어기는 것을 봐도 침묵하는 편이다. → 전혀 아니다 / 무관심은 조직의 윤리성을 해치는 행동입니다.
              109. 내가 한 실수로 팀에 손해가 생기면 먼저 책임을 밝힌다. → 매우 그렇다 / 솔직함과 신뢰 구축 태도입니다.
              110. 팀원에게 피해를 줄 수 있는 행동은 철저히 피하려 한다. → 매우 그렇다 / 조직 공동체의 일원으로서 책임 의식입니다.
              111. 회사의 이름으로 일하는 만큼 평판을 항상 의식한다. → 매우 그렇다 / 기업 대표로서 윤리와 신뢰 관리가 중요합니다.
              112. 나는 공정한 경쟁이 가능한 환경을 스스로 만들려고 한다. → 매우 그렇다 / 정의롭고 투명한 문화 형성을 위한 자세입니다.
              113. 나는 나의 언행이 후배들에게 미칠 영향을 고려한다. → 매우 그렇다 / 조직 내 본보기로서 책임 있는 태도입니다.
              114. 실수를 인정하는 것이 리더십의 일부라고 생각한다. → 매우 그렇다 / 겸손한 태도는 신뢰를 높이는 리더의 조건입니다.
              115. 내가 책임지지 않아도 되는 일에도 적극적으로 관여할 때가 있다. → 매우 그렇다 / 조직 기여와 협업 의지를 반영합니다.
              116. 사소한 것도 정확하게 전달하려고 한다. → 매우 그렇다 / 커뮤니케이션의 신뢰성과 정직성을 보여줍니다.
              117. 회사의 비전이나 철학에 공감할 수 있어야 최선을 다할 수 있다. → 매우 그렇다 / 가치관의 일치가 조직 몰입도에 영향을 줍니다.
              118. 잘못된 보고나 기록을 바로잡는 것이 귀찮을 때도 있다. → 전혀 아니다 / 투명성과 정직은 기본 중의 기본입니다.
              119. 조직의 평판보다 개인의 성과가 더 중요하다고 생각한다. → 전혀 아니다 / 조직 공동체로서 책임 있는 태도가 요구됩니다.
              120. 단기 성과를 위해 원칙을 양보하는 것은 옳지 않다고 생각한다. → 매우 그렇다 / 윤리와 원칙은 결과보다 우선시되어야 합니다.

              대기업 인성검사의 채점 핵심 원칙 3가지 #

              1. 일관성 Consistency

                • 검사 내에서 유사한 의미의 질문을 다양한 형태로 반복하여 일관성을 확인: 같은 사람이라면 비슷한 질문에 비슷한 답을 해야 신뢰도 있는 응답으로 간주된다.
                • 예시
                  • 문항 A: “나는 협업하는 것을 좋아한다.” → ‘매우 그렇다’ 문항 B: “나는 혼자 일하는 것이 더 편하다.” → ‘매우 그렇다’ → 모순
                  • 문항 A: “나는 새로운 일에 도전하는 것을 두려워하지 않는다.” → ‘그렇다’ 문항 B: “나는 변화보다 익숙한 일이 좋다.” → ‘그렇다’ → ❌ 의미 충돌
                • 채점 방식
                  • 문항쌍 간 상관관계 및 내적 신뢰도(Cronbach’s alpha) 활용
                  • 앞뒤 문항의 응답이 모순되면 감점 또는 비정상 응답으로 간주

              2. 진실성 Sincerity

                • 검사 응답자가 문항에 대해 솔직하고 일관된 자기 본래의 생각을 표현했는가를 평가
                  • 자기모순: 같은 내용의 문항에 반대되는 응답을 함
                  • 무성의 반응: 아무 생각 없이 무작위 또는 반복적 응답 (모든 문항에 ‘보통이다’ 또는 동일한 값만 체크)
                  • 의도된 왜곡: 좋은 평가를 의식해 자신의 성향을 과도하게 이상화 (“나는 절대 화를 내지 않는다”, “나는 항상 모든 사람을 이해한다” → 매우 그렇다 (비현실적))
                • 예시
                  • “나는 항상 모든 업무를 완벽하게 처리한다.” → 현실적으로 누구나 실수 가능
                  • “나는 절대 감정적으로 행동하지 않는다.” → 너무 이상적이면 오히려 비현실적
                • 채점 방식
                  • 채점 방식
                    • 이상화 지향 척도(Social Desirability Index)를 통해 → “자신을 얼마나 과도하게 이상적으로 보이게 하려 하는가"를 측정
                    • 점수가 너무 높거나 낮으면 탈락 요인

              3. 이상성/극단성 Desirability & Extremeness

                • 모든 문항에 ‘매우 그렇다’ 또는 ‘전혀 아니다’로 답하는 경우 → 정상적인 사람의 성향 분포와 다르다고 판단
                • 바람직한 사람의 응답 경향
                  • 대부분의 사람은 중간~약간 긍정/부정이 섞인 형태로 응답함
                  • 실제 인성검사 알고리즘은 정규분포에 가까운 응답 패턴을 “자연스럽고 신뢰할 수 있는 응답"으로 간주함.
                • 채점 방식
                  • 채점 방식: 응답의 표준편차, 평균, 치우침(skewness) 등을 계산 → 너무 치우친 경우 ‘극단 반응’ 경향으로 페널티

              이상화 성향(desirability) 판단 문항 50개 #

              아래 문항에 모두 ‘매우 그렇다’로 응답하면 현실성 부족 또는 과장된 응답으로 간주되어 감점될 수 있음.

              [1] 자기 통제 과잉 탐지

              1. 나는 어떤 상황에서도 감정을 전혀 드러내지 않는다.
              2. 나는 화가 나더라도 상대에게 표현하지 않는다.
              3. 나는 스트레스를 절대 받지 않는다.
              4. 나는 항상 냉정하고 이성적으로 행동한다.
              5. 나는 어떤 유혹에도 흔들리지 않는다.
              6. 나는 결정을 후회한 적이 거의 없다.
              7. 나는 단 한 번도 충동적으로 행동한 적이 없다.
              8. 나는 실수를 해도 전혀 당황하지 않는다.
              9. 나는 불공정한 상황에서도 항상 침착하게 대응한다.
              10. 나는 피곤하거나 지쳐도 감정을 숨긴다.

              [2] 과도한 도덕성/정직성 강조

              1. 나는 거짓말을 한 적이 없다.
              2. 나는 어떤 상황에서도 규칙을 어긴 적이 없다.
              3. 나는 타인의 물건에 손을 댄 적이 단 한 번도 없다.
              4. 나는 약속을 어긴 적이 한 번도 없다.
              5. 나는 양심에 어긋나는 행동을 한 적이 없다.
              6. 나는 실수를 하면 항상 즉시 고백한다.
              7. 나는 작은 이익이라도 부정하게 얻지 않는다.
              8. 나는 지각이나 조퇴를 한 적이 거의 없다.
              9. 나는 타인에게 절대 거칠게 말한 적이 없다.
              10. 나는 도덕적으로 항상 모범적인 삶을 살아왔다.

              [3] 과잉된 이타심 강조

              1. 나는 항상 남을 먼저 배려한다.
              2. 나는 내 감정보다 타인의 감정을 더 우선시한다.
              3. 나는 내 일을 미루고서라도 남을 돕는다.
              4. 나는 도움을 요청받으면 절대 거절하지 않는다.
              5. 나는 항상 팀원들의 기분을 먼저 살핀다.
              6. 나는 내 실적보다 팀의 성과가 더 중요하다고 생각한다.
              7. 나는 남이 잘되면 질투 없이 기쁘다.
              8. 나는 모두가 행복해질 수 있는 방향을 항상 고민한다.
              9. 나는 팀원들의 일을 자주 대신 처리한다.
              10. 나는 모두가 좋아하는 사람이라는 말을 자주 듣는다.

              [4] 완벽주의/자기 이상화

              1. 나는 늘 철저하게 계획하고 실행한다.
              2. 나는 한 번도 일을 미룬 적이 없다.
              3. 나는 어떤 일이든 100% 해내려고 한다.
              4. 나는 한 번도 마감기한을 넘긴 적이 없다.
              5. 나는 항상 최상의 결과를 목표로 삼는다.
              6. 나는 맡은 일에서 한 번도 실수한 적이 없다.
              7. 나는 누구보다 성실하게 일해왔다고 자부한다.
              8. 나는 항상 정확하고 완벽한 보고서를 쓴다.
              9. 나는 내 업무에 대해 타인의 지적을 받은 적이 거의 없다.
              10. 나는 모든 상황에서 최선을 다했다.

              [5] 자기 이미지 과포장

              1. 나는 항상 긍정적인 마인드를 유지한다.
              2. 나는 누구와도 갈등 없이 지낸다.
              3. 나는 스트레스를 받아도 표정 변화가 없다.
              4. 나는 항상 조직 내에서 모범적인 구성원이다.
              5. 나는 부정적인 말을 거의 하지 않는다.
              6. 나는 상사에게 한 번도 혼난 적이 없다.
              7. 나는 조직 내에서 갈등의 원인이 된 적이 없다.
              8. 나는 어떤 상황에서도 예의 바르게 행동한다.
              9. 나는 내 잘못으로 팀에 피해를 준 적이 없다.
              10. 나는 나에 대해 부정적인 피드백을 들어본 적이 거의 없다.

              [6] 타인에 대한 과도한 포용성 강조

              10. 나는 내 행동이 조직 전체에 영향을 미친다고 항상 생각한다.

              [10] 현실 도피성 긍정 강조

              1. 나는 업무 중 힘들다고 느낀 적이 없다.
              2. 나는 직장에서 단 한 번도 스트레스를 받은 적이 없다.
              3. 나는 항상 출근이 기대된다.
              4. 나는 직장에서 실망하거나 불만을 느껴본 적이 없다.
              5. 나는 업무 피로를 느껴도 항상 활기차게 행동한다.
              6. 나는 주말보다 평일이 더 즐겁다.
              7. 나는 매일 아침 일을 시작하는 게 설렌다.
              8. 나는 회의, 야근 등 어떤 업무도 전혀 부담스럽지 않다.
              9. 나는 업무량이 많아도 전혀 불평하지 않는다.
              10. 나는 직장에서 단 하루도 무기력했던 적이 없다.

              해설

              1. “나는 새로운 환경에 잘 적응한다"면서 “변화에 스트레스를 받는다"고 하면 변화 수용에 대한 자기 인식이 모순됩니다.
              2. “새로운 방식을 선호한다"면서 “새로운 업무에 부담을 느낀다"고 하면 도전에 대한 태도가 일관되지 않습니다.
              3. “변화는 긍정적이다"면서 “낯선 상황이 불편하다"고 하면 말과 감정 반응이 충돌합니다.
              4. “낯선 사람과 협업 가능하다"면서 “새로운 시스템은 피한다"고 하면 개방성과 회피가 동시에 나타나 모순입니다.
              5. “도전을 즐긴다"면서 “루틴한 업무가 편하다"고 하면 업무 선호에서 혼란이 있습니다.
              6. “변화에 거부감이 없다"면서 “계획된 방식 외엔 시도하지 않는다"고 하면 실천과 인식이 어긋납니다.
              7. “변화가 흥미롭다"면서 “불확실한 상황을 피한다"고 하면 태도에 일관성이 부족합니다.
              8. “위기에 유연하다"면서 “갑작스러운 변화에 당황한다"고 하면 스트레스 대응에서 모순입니다.
              9. “도전 과제를 좋아한다"면서 “안정적인 일만 맡고 싶다"고 하면 동기 체계가 상충됩니다.
              10. “새로운 분야를 배우려 한다"면서 “예측 가능한 일이 좋다"고 하면 학습 태도와 안정 추구가 충돌합니다.
              11. “협업을 좋아한다"면서 “혼자 일하는 게 더 효율적"이라고 하면 사회적 성향이 불일치합니다.
              12. “소통을 중시한다"면서 “혼자 결정하는 게 편하다"고 하면 협업 태도가 일관되지 않습니다.
              13. “타인의 의견을 존중한다"면서 “내 방식이 최고다"라고 하면 수용성과 고집이 충돌합니다.
              14. “의견을 듣는 게 중요하다"면서 “내가 옳으면 반대를 무시한다"고 하면 리더십 스타일에서 모순입니다.
              15. “일을 끝까지 책임진다"면서 “어려워지면 포기하고 싶다"고 하면 책임감에 대한 자기평가가 모순됩니다.
              16. “기한을 철저히 지킨다"면서 “일을 자주 미룬다"고 하면 실행력에 신뢰가 떨어집니다.
              17. “최선을 다한다"면서 “결과가 나쁘면 남 탓하고 싶다"고 하면 책임 태도가 불일치합니다.
              18. “시간을 철저히 관리한다"면서 “마감을 자주 넘긴다"고 하면 계획성과 행동이 불일치합니다.

              21. “실수는 솔직히 말한다"면서 “숨긴 적이 있다"고 하면 정직성에 자기모순이 드러납니다.
              22. “부정은 절대 안 쓴다"면서 “편법을 쓰고 싶을 때가 있다"고 하면 윤리 기준이 흔들립니다.
