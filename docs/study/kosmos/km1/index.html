                    KOSMOS 인수인계 #

                    #2026-04-08

                    오늘 KOSMOS 프젝의 인수인계를 받았는데 주로

                    1. 회의자료 및 회의록 작성
                    2. eCRF에 업로드된 파일 기반의 VCF, WGS summary 파일 작성

                    정도인것같고 관련 코드들이랑 인수인계 파일을 정리해볼려구한다

                    #

                    1. VCF 회의
                    • 레포트 추출을 eCRF에 올리는데 한달에 한번씩 다른 병원이 올린 정보를 확인해서 전달하는 회의.
                    • 160>1 회의록
                    • 160>2 회의자료
                    • 회의 날짜는 한달에 한번, 달에 두번째주 월요일인데 이번에는 5월 11일로 진행
                    • ncdc 공동계정으로 줌링크를 회의자료랑 같이 송부. 회의 끝나면 바로 메일.

                    #

                    1. VCF 회의자료 생성
                    • 160>2 회의자료 에서 ‘날짜_회의자료.hwp’를 만들면됨
                    • panel.seq VCF 파일 업로드 현황 / WGS(VCF) 파일 업로드 현황 / VCF 파일 수집 및 이슈 현황을 작성
                    • 회의자료 완성하면 이니셜 넣어서 지영쌤께 수정다했다고 말씀드리기 회의자료 완성된걸로 줌링크랑 같이 메일 송부.
                    • 160>14 에서 하는법 확인하기.

                    #

                    1. VCF summary 작성
                    • 160>11>1에 한달치 vcf를 넣어둔다.
                    • eCRF에서 DataTool을 다운받고 CRF_INSDT, CRF_UPDDT 컬럼을 확인해서 이번달 신규/수정 파일을 확인한다.
                    • eCRF에서 신규 VCF를 다운받고 파일명변경 후 신규 디렉토리에 넣어준다.
                    • 160>14>vcf 구축메뉴얼 보고 하기.
                      • 한달치를 자료추출해서 다운받을수잇고
                      • 기관별 summary에 crp_upddt 보고 list 시트에 신규 적어주기. f 컬럼 사용해서 드래그해주면 함수로 자동으로 들어간다. vcf 파일이름은 ngs file 이름을 넣기. 신규 아니고 변경이면 b 컬럼에 변경 적고 노란색으로 대상자 id 마킹
                      • vcf 구축메뉴얼. p8처럼 피벗 테이블로 대상자수 파악. 새 시트 → location / name을 행 값에 넣어준다.
                      • 기관별 summary에 시트 새로 만들어서 넣어준다 오름차순 아니고 sub id 순으로
                      • 20260331 summary → 파일명 → 파일명 없음.
                      • 업로드 파일 수에다가 대상다 아이디를 쭉 넣어준다.
                      • 11p- pdf 파일로 준 경우 확장자 변경 재요청한다.
                    • eCRF 자료추출 하는법
                      • data tools → 자료 추출. 자료추출 추가 > 데이터셋 생성
                      • 대상자등록현황도 똑같이 추출.
                      • vcf 는 ecrf에 들어가서 대상자 id 기관 치면 뜬다. nts 보면 vcf 파일.
                    • 공지사항 작성
                      • 17p 공지사항 작성 > 내 파트만 변경. 160>10>3 > 공지사항 pdf 보면 거기에 vcf 만 붙여주면된다. vcf 파일 재송부 필요 등등 쓰기.

                    #

                    1. WGS VCF
                    • wgs vcf는 메일로 zip으로 오니까 다운받기. 160>10>12 에 넣기. 6월에 받는다.
                    • wgs 구출메뉴얼. 박주영 선생님께 요청을 드리면 되고 다운받은게 018_wgs에 들어잇다.
                    • 반출 목록 파일 만들때 코드 돌린다. 파일명 변경한 뒤에 summary 시트에 변경된 파일명 붇이기.
                    • wgs 구축메뉴얼에 10p는 무시
                    • 160>10>12>날짜 wgs> 파일을 코어가 들어오면 옮겨준다.
                    • 160>11>1>날짜>vcf 이상여부확인 리스트가 파이썬 돌리면 그냥 나온다.
