                            DBSCAN #3 MutClust 슈도코드 #

                            #2025-07-28

                            1 #

                            Input:
    - D: 데이터 포인트 집합
    - eps: 이웃 거리 임계값
    - minPts: 최소 이웃 수 (밀도 기준)

Output:
    - cluster_labels: 각 데이터 포인트에 대한 클러스터 라벨 (노이즈는 -1)

Initialize:
    - cluster_id ← 0
    - label[x] ← UNVISITED for all x in D

                            데이터 집합 D, 파라미터 eps와 minPts가 들어간다.

                            2. H-중요도 계산 #

                            For each point x in D:
    H ← calculateHscore(x)

                            각 데이터포인트에 대해 H-score를 계산한다.

                            3. 임시 Eps 계산 및 후보 Core 돌연변이 탐색 #

                            For each point x in H:

    eps ← calculateEps(x, Escaler)               // x의 임시 eps 계산 (H 점수 기반)           
//  Hsum, Hmean, MtCount, Neighbors ← regionQuery(x, eps) 

//  If Hsum ≥ 0.05 and Hmean ≥ 0.01 and H(x) ≥ 0.03 and MtCount ≥ 5:
//      label[x] ← CCM

                            현재위치 x의 H score 기반으로 임시 eps를 계산한다.

                            For each point x in H:

//  eps ← calculateEps(x, Escaler)                 
    Hsum, Hmean, MtCount, Neighbors ← regionQuery(x, eps)   // eps 내 이웃 점수 분석

    If Hsum ≥ 0.05 and Hmean ≥ 0.01 and H(x) ≥ 0.03 and MtCount ≥ 5:
        label[x] ← CCM  // 조건 만족 시 CCM(Core Cluster Member)로 라벨링

                            그리고 regionQuery로 eps 내 돌연변이들의 중요도를 확인한다. 조건을 만족하면, CCM(Candidate Core Mutation)으로 처리한다.

                            Function regionQuery(x, eps):
    neighbors ← ∅

    For each point y in H:
        If distance(x, y) ≤ eps:
            neighbors ← neighbors ∪ {y}

    Hsum ← sum(score of y in neighbors)
    Hmean ← sum_scores / |neighbors|
    MtCount ← len(neighbors)

    return (Hsum, Hmean, MtCount, neighbors)

                            4. 클러스터 확장 (CCM일 경우) #

                            For each point x in H:

//  eps ← calculateEps(x, Escaler)             
//  Hsum, Hmean, MtCount, Neighbors ← regionQuery(x, eps) 

//  If Hsum ≥ 0.05 and Hmean ≥ 0.01 and H(x) ≥ 0.03 and MtCount ≥ 5:
//      label[x] ← CCM 

        cluster_id ← cluster_id + 1
        expandCluster(x, N, ClusterID, eps, minPts, label)

                            CCM일 경우 expandCluster를 사용해서 클러스터 확장을 수행한다.

                            Function expandCluster(x, neighbors, ClusterID, eps, minPts, label, DiminFactor):
    label[x] ← ClusterID
    Idx ← 0

    FinalNeighbors ← ∅   // 최종적으로 클러스터에 포함할 점들

    For each point y in neighbors:
        Idx ← Idx + 1
        dynamic_eps ← int(eps - (Idx / DiminFactor))

        If dynamic_eps < Idx:
            break

        If label[y] = UNVISITED:
            label[y] ← ClusterID
            N_prime ← regionQuery(y, dynamic_eps)

            If |N_prime| ≥ minPts:
                neighbors ← neighbors ∪ N_prime  // core point 이웃 확장

            FinalNeighbors ← FinalNeighbors ∪ {y}

        Else If label[y] = NOISE:
            label[y] ← ClusterID  // noise → border point로 재분류
            FinalNeighbors ← FinalNeighbors ∪ {y}

    return FinalNeighbors

                            노이즈였지만 현재는 x의 이웃포인트가 된 y는 border point로 재분류해준다.

                            #

                            Function calculateEps(x, Escaler):
    return x * Escaler

                            For each point x in H:

//  eps ← calculateEps(x, Escaler)               // x의 임시 eps 계산 (H 점수 기반)
//  Hsum, Hmean, MtCount, Neighbors ← regionQuery(x, eps)   // eps 내 이웃 점수 분석

//  If Hsum ≥ 0.05 and Hmean ≥ 0.01 and H(x) ≥ 0.03 and MtCount ≥ 5:
//      label[x] ← CCM  // 조건 만족 시 CCM(Core Cluster Member)로 라벨링

//  eps ← calculateEps(x, Escaler)  // x의 임시 eps 계산
//  N ← regionQuery(x, eps)   // x 주변의 eps 이내 이웃 포인트 탐색
  
    If |N| > minPts:
        label[x] ← CCM

      cluster_id ← cluster_id + 1
      expandCluster(x, N, cluster_id, eps, minPts, label)
    Else:
        label[x] ← UNVISITED
