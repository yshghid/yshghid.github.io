    ORDER BY #

    order by는 데이터를 정렬시킨다.

    SELECT * FROM NETFLIX ORDER BY REG_DATE ASC;

    -> asc 오름차순이 기본값이라서 생략해도 된다.

    역순으로 정렬

    SELECT * FROM NETFLIX ORDER BY REG_DATE DESC;
