                ‘컴퓨터과학’의 개념? #

                2025-01-09 #

                과학은 자연과학, 형식과학, 응용과학, 사회과학으로 나뉜다. 형식과학에는 논리학, 수학, 시스템 과학, 전산학, 통계학 등이 속해있다.

                전산학의 경우 현재는 컴퓨터과학, 컴퓨터공학이라는 표현이 보편적이다. 컴퓨터과학(Computer Science)은 응용 수학, 컴퓨팅 이론, 문제를 다루는 분야이며, 컴퓨터공학(Computer Engineering)은 컴퓨터과학 분야 중에서도 하드웨어를 다루는 세부 영역이다.

                세부 분야 #

                컴퓨터과학의 세부 분야는 다음과 같다.

                1. 이론 컴퓨터 과학 / 수학
                • 이론 컴퓨터 과학 - 확률론 및 통계학, 선형대수학, 이산수학(자료구조, 알고리즘, 정보이론, 그래프이론, 수리논리학)
                • 수학기초론
                • 이산수학
                1. 하드웨어
                • 전자공학
                • 컴퓨터 구조론
                • 신호 처리 및 제어
                • 임베디드 컴퓨팅/마이크로프로세서
                • 컴퓨터 하드웨어 응용: 양자컴퓨터/뉴로모픽 컴퓨팅/인공지능 가속기
                1. 소프트웨어
                • 소프트웨어 개발방법론
                • 계산과학 - 생물정보학, 수치해석학, 계산생물학/물리학/화학/신경과학
                • 컴퓨터 그래픽스
                • 운영체제
                • 데이터베이스 - 데이터베이스, 데이터 마이닝, 빅데이터, 검색 엔진, 블록체인
                • 통신 및 컴퓨터 네트워크
                • 정보보안
                • 인공지능/신경과학 - 인공지능, 인공신경과학, 계산학습이론, 계산신경과학, 유전 알고리즘, 뉴로모픽 컴퓨팅
                • 자연어 처리
                • 프로그래밍
                • 금융공학
                • HCI(인간-컴퓨터 인터렉션)(Human-Computer Interaction)

                나의 관련분야 #

                운영체제 #

                운영체제는 사용자가 컴퓨터를 관리하는 인터페이스이며 용도에 따라 임베디드, 서버, 모바일, 데스크탑 운영체제가 있다. 서버용 운영체제는 Linux, UNIX, Windows, macOS 등이 있다. macOS 운영체제를 사용함.

                언어 #

                프로그래밍 언어는 사람이 원하는 작업을 컴퓨터가 수행할 수 있도록 전달하는 역할을 하며 python, R, Javascript, C++, MATLAB 등이 있다. python, R을 사용함.

                알고리즘(군집 분석) #

                알고리즘의 정의는 문제를 ‘효과적으로’ 해결하기 위한 절차나 방법이다. 자료구조의 스택, 큐, 힙, 트리, 정렬, 탐색(DFS, BFS) 알고리즘을 포함해서 여러 분야의 알고리즘이 존재한다. 인공지능 분야의 경우 기계학습에서 인공신경망, SVM(서포트 벡터 머신), 데이터 마이닝에서 군집 분석, 회귀 분석 등의 알고리즘이 있다.

                군집 분석(cluster analysis)는 각 데이터의 유사성을 측정하여 다수의 군집으로 나누고 군집 간의 상이성을 확인하는 분석이다. 개념 자체는 통계학의 관점에서 다변량 분석(multi-variate analysis)의 한 종류로서 이전부터 존재하는 개념이었지만 인공지능 및 기계학습의 발전과 함께 급격히 체계화되었다.

                군집 분석은 계층적 군집화, 분할적 군집화로 나뉜다. 분할적 군집화는 서로 계층적 관계가 없는 다수의 군집을 만드는 방법으로, 중심점 기반 군집화(center-based clustering), 밀도 기반 군집화(density-based clustering), 확률 기반 군집화(probability-based clustering), 분포 기반 군집화(distribution-based clustering), 그리고 그래프 기반 군집화(graph-based clustering)가 있다.
