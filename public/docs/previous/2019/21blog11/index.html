    살만한이유 #

    #2021-06-05 01:48:32

    나는지금이 제일좋다 좋은걸 느끼진못하는데 이성적 사고를따라가보면 좋은걸 알수있다. 예전에 싫었던것들 지금보면 거의다 삭제돼있고 예전에 갖고싶었던것들 다는 아니어도 조금은갖췄고 갖추고싶은것 아직 남았고…

    길가다 창에비친모습, 매일똑같은데도 사무치는 고통. 내가 태어날때부터 갖지못한것과 그걸가진누군가를 보는나 그냥 세상속에놓인 나를 인지하는 기타등등 사소한상황속에서 나는 소소히 하지만 확실한불행을 느낀다. 근데 그것들이 있음이오히려 안정감으로 다가오니까괜찮다 왜냐면 기분이 어느정도로 유지되는지 인지하고살수있는게 좋기때문이다. 안정감은좋은것이다 작은불행도 작은행복도 결국은예측할수있고 손안에 있다는게 중요한거다 뭐가됐든… 내손안에 있다는게 의미가있다
